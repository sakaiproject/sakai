Home/Site Information Display: Overview

What it does

On the Home page of each course site or project site, you will see areas for general information (e.g., Site Information Display) and synoptic views for communication tools (e.g., Recent Announcements, Messages and Forums Notifications, and Recent Chat Messages) used in your site.

Key concepts

Content options: You can display a web page or text in your Site Information Display box. For instructions, see Adding content to the Site Information Display box.

Messages and Forums Notifications: A summary of all new and unread messages that exist in the site. For more information, see Messages and Forums notifications.

Things to consider

  • Whatever you display inside Site Information Display will be viewable by everyone with access to your site.
  • Instructors are visible within a course site by default.
