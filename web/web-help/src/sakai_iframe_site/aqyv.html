Home/Site Information Display: Overview

What it does

On the Home page of each site, you will see areas for general information (e.g., Site Information Display) and synoptic views for communication tools (e.g., Recent Announcements, Recent Chat Messages, and notifications of unread messages and forums) used in your site.

Key concepts

Content options: You can display a web page or text in your Site Information Display box; see Editing content in the Site Information Display box.

Messages and Forums Notifications: You can display a summary of all new and unread messages that exist in the site; see Messages and Forums notifications.

Things to consider

  • Anything you display inside Site Information Display will be viewable by everyone with access to your site.
  • Instructors are visible within a course site by default.
