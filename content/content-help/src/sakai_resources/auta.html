Undoing an ending date to make a Resources item visible again

You can set beginning and ending dates for items in your site's Resources. If an item with an ending date is no longer visible, and you'd like to make it available again, follow these steps:

  1. In your site's menubar, click Resources.

  2. Find the file you'd like to make available, and next to its name, from the Actions menu, choose Revise Details.

  3. In the window that opens, under "Availability and Access", select Show. Also, you can reset beginning and ending dates, if desired.

  4. When you're finished, click Update.
