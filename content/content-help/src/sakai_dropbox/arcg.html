Adding a new folder

By default, your Drop Box has one folder in it containing your name. To add a subfolder:

  1. Next to the existing folder, from the Add menu, choose Create Folders.

  2. Type a name for the folder in the box provided.

  3. To enter an optional description, click Add details for this item.

  4. You can add additional folders from this screen by clicking Add Another Folder.

    Note: To remove a file from the list of files to add, click the red X next to it.

  5. Click Create Folders Now.

Adding a new Drop Box item

When you add a new Drop Box item, you may have the option of sending an email notification. This option can be enabled or disabled by the site instructor/owner.

Instructors will see an orange star highlighting Drop Box folders that have recently been updated.

Follow the instructions below for the type of item you want to add.

File upload

To upload an existing document or other file from your computer:

  1. Next to the folder to which you want to upload the file, from the Add menu, select Upload Files.

  2. For each item, click Browse, select a file to upload, and then click Open.

    Note: Filenames that include certain characters cannot be accepted. For a list of unsupported characters, see Unsupported characters.

  3. Next to "Display Name", type a title for the item (if you don't add a title, the filename will be used for the title).

  4. If desired, click Add details for this item, which will allow you to enter a description, choose a copyright status and add copyright information, and add detailed descriptors by clicking Optional properties (see the Optional properties section below for more information).

  5. To upload more than one file, click Add Another File.

    You can add any number of files at once, as long as the total file size does not exceed 100 MB.

    Note: To remove a file from the list of files to add, click the red X next to it.

  6. Click Upload Files Now to finish.

URL

To specify a URL to a web site as the Drop Box item:

  1. Next the folder to which you want to add the URL, from the Add menu, select Add Web Links (URLs).

  2. For each item, type the URL in the box. (You may omit the http:// prefix.)

  3. Add a title for your URL link item. If you don't add a title, the URL will be used as the title of the item.

  4. If desired, click Add details for this item, which will allow you to enter a description and add detailed descriptors by clicking Optional properties (see the Optional properties section below for more information).

  5. To add another URL, click Add Another Web Link.

    Note: To remove a URL from the list of URLs to add, click the red X next to it.

  6. Click Add Web Links Now to finish.

HTML page

To create and display an HTML document:

  1. Next to the folder to which you want to add the HTML page, from the Add menu, select Create HTML Page.

  2. Enter the content of your page and click Continue.

  3. Enter a title and an optional description in the appropriate text areas.

  4. Choose your page's copyright status, add copyright information, and choose whether or not to display a copyright alert.

  5. To add detailed descriptors, click Optional properties. See the Optional properties section below for more information.

  6. Click Finish to add your HTML page to Drop Box.

Text document

To create and display a plain text document:

  1. Next to the folder to which you want to add the document, from the Add menu, select Create Text Document.

  2. Type the content of your text document and click Continue.

  3. Enter a title and optional description in the appropriate text areas.

  4. Choose your text document's copyright status, add copyright information, and choose whether or not to display a copyright alert.

  5. To add detailed descriptors, click Optional properties. See the Optional properties section below for more information.

  6. Click Finish to add your text document to Drop Box.

Editing a Drop Box item

To edit the name, description, or copyright status, choose Actions, and then click Edit Details. Make your changes, and then click Update to save them.

To update the content of most items, choose Actions, and then click Edit Content. Make your changes, and then click Continue. For a file, you will have to make the changes and save them on your computer, and then upload a new version. From the Actions menu, select Upload New Version.

Deleting a Drop Box item

To delete an item from Drop Box:

  1. To remove one item at a time, choose Actions, and then click Remove. To remove several items at a time, mark their corresponding checkboxes, and then, above the list of items, click Remove.

  2. To confirm that you want to remove the item(s) listed, click Remove. Or, to cancel, click Cancel.

Optional properties

To add detailed descriptors, click Optional properties. You can then add information to the following fields:

  • Alternate Title
  • Creator
  • Publisher
  • Subject and Keywords
  • Date Created
  • Date Issued
  • Abstract
  • Contributor
  • Audience
  • Audience Education Level
