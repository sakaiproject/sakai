Configuració de l'eina de Contingut Web

Feu servir aquesta eina per mostrar una pàgina web a l'espai.

Feu clic a Opcions i introduïu l'adreça de la pàgina web al camp "Adreça URL".

Per exemple, per fer que aquesta eina mostri la pàgina de cerca de Google, heu d'escriure "www.google.com" al camp "Adreça URL" del formulari "Opcions".
