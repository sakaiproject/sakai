NOTES:
The [BLOCKED] page indicates that the ADL Sequencer hit a point in the conceptual activity tree that it could no longer traverse and the candidate activity was not deliverable.

Typical reasons this page is being encountered include:
  • A Start Navigation Request has failed to identify an available leaf activity.
  • A Continue or Previous Navigation Request has encountered a disabled activity. Activities can become ‘disabled’ from any of the following reasons:
    • Limit Condition violation
    • Disabled Precondition Sequencing Rule
  • A Choice Navigation Request has targeted a cluster that does not have an available descendent leaf activity.
  • A Choice Navigation Request has targeted an activity that “constrained” relative to the Current Activity.
