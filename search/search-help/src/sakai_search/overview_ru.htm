Поиск: Обзор

Инструмент "Поиск" позволяет по заданным параметрам искать содержимое, созданное с помощью различных инструментов, установленых на сайте курса или проекта. Это делается с помощью составления словарного указателя (индексирования) всего содержимого сайта, которое доступно для инструмента "Поиск". Словарный указатель автоматически обновляется: как только на сайт добавляется новое содержимое, оно тут же индексируется.

Например, если на сайте заведен список почтовой рассылки, все сообщения, рассылаемые через него, индексируются и становятся доступными для поиска по ним с помощью инструмента "Поиск".

Поисковая машина индексирует информацию, представленную по-разному, в том числе сообщения, веб-страницы в формате HTML, документы в формате DOC, PDF илиRTF, электронные таблицы в формате XLS и страницы совместного редактирования Wiki.

Как искать

На странице инструмента "Поиск" вы найдете поле для текстового ввода. Просто введите в него слова, которые хотите найти, и нажмите на кнопку "Поиск".

Использование RSS для просмотра результатов поиска

Выполнив поиск, вы обнаружите иконку RSS, появившуюся вместе с результатами поииска. Это позволяет вам просмотреть результаты поиска с помощью вашей любимой программы для просмотра RSS. Если вы потратите немного времени, чтобы освоить расширенный поиск, то обнаружите, что вы можете запускать RSS-каналы с любыми видами информации в качестве ключевых слов.
