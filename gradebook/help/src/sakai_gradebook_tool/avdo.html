Importing grades for existing Gradebook items

The Gradebook allows you to import grades for existing Gradebook items, as well as items that have not yet been created. To import grades and create a new Gradebook item, see Importing grades from a spreadsheet. To import grades for existing Gradebook items:

  1. In your site's menubar, click Gradebook.

  2. Near the top, click Import Grades.

  3. Download the spreadsheet template by clicking Download Spreadsheet Template.

  4. Edit the spreadsheet in your favorite spreadsheet application, such as Excel, and save it as a .csv file.

  5. Import the spreadsheet by clicking Browse... and choosing the .csv file.

  6. After selecting the file, click Import Spreadsheet.

  7. The following page should show the imported grades. When you are finished, click OK.

Things to consider

  • You cannot import grades for Gradebook items sent from the Assignment tool.

  • Grades imported using the Import Grades option will overwrite any pre-existing grades. Always be sure to use the spreadsheet template downloaded in step 3.
