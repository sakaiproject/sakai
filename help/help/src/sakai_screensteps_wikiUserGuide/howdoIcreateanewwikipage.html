How do I create a new wiki page?

To create a new page, you will need to add it as a link from an existing page.

Go to Wiki.

Select the Wiki tool from the Tool Menu of your site.

Click Edit.

While viewing an existing page, click the Edit button.

Enter the page name.

In the editing area, type the name of the page in brackets, and then click Save. For example, typing [New Page] will create a page titled "New Page".

Note: You can't use the following characters in an Wiki link or title:

: Colon

@ At

# Hash

| Pipe

\ Back-Slash
