How do I make wiki pages available to groups?

Go to Wiki.

Select the Wiki tool from the Tool Menu of your site.

Click Edit.

Edit the page where you would like the links to the group pages to appear.

Create a new page for each group.

  1. Use the wiki markup to create a page for each group. To do this, type the name of the new page you want to create in square brackets, for example: [Group 1 Page].
  2. Click Save to save the new page links.

Click on one of the group pages to edit that page.

Edit the group page.

Select the group or groups with access to the page.

At the bottom of the page, complete the following steps:

  1. Select the Display this page to selected Groups only radio button.
  2. Choose a group from the drop-down menu.
  3. Click Save.

Note: You must have existing groups in your site in order to see the group options.

Repeat the process for any additional group pages. (Optional)

View the group pages in the wiki.

Any pages which are restricted to specific groups will display a message indicating that they are only available to the selected group/s.

Users who are not members of the selected group will not see the page link at all.
