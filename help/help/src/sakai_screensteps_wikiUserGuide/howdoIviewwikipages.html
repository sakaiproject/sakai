How do I view wiki pages?

To move from page to page in Wiki, follow the links within each page. Links with a world icon beside them link to external websites or to attachments. These will open in a new window.

Unlike some wiki tools, Wiki does not have a separate menu to navigate through the pages. All navigation must be done through links on the pages.

Go to Wiki.

Select the Wiki tool from the Tool Menu of your site.

Click on Home link to return to the top level wiki page.

Click on edit to create a new page

To create a new page type the name of the new page you want to create in square brackets: [my new page]

If you have navigated down one or more levels via page links and need to return to the Home page to navigate to another area, use the Home link in the top menu to return to the top level of the wiki.

Click on the page links to navigate through wiki content.
