What are External Tools?

If you're a functional administrator, External Tools lets you configure integration with an external solution that utilizes LTI, an independent standard for web-based applications developed by IMS. You can integrate solutions that are compliant with either of the following LTI standards:

  • LTI 1.1
  • LTI 1.3 or LTI Advantage

You can make the external tool available in a specific site or in all sites. You may also choose where the external tool is available within the Sakai UI. Depending on how the external tool availability has been configured, site owners can add the tool via Lessons, Assignments, Rich Text Editor, or to the Tool Menu site navigation via Site Info (using Manage Tools or External Tools).

External Tools lets you determine tool configuration options for site owners. You may provide no options, so site owners can only add the external tool. Or, you may allow site owners to change a variety of parameters, such as the tool name that displays in a site's Tool Menu or the tool's frame height.

To access this tool, select External Tools from the Tool Menu of the Administration Workspace.
