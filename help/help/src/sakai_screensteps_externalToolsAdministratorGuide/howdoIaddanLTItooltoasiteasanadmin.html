How do I add an LTI tool to a site as an admin?

This article shows how to add a single external tool link so that it appears in the left navigation menu of an individual site. You may use this process to create individual site navigation links to regular LTI tools, or stealthed (i.e. hidden or restricted access) LTI tools. For information about deploying stealthed tools to multiple sites at once, please see How do I add a Stealthed LTI tool to one or more sites?

Go to External Tools.

Select the External Tools link from the Tool Menu in the Administration Workspace.

Click Tool Links.

Note: Only previously installed tools will appear in this list. If the LTI tool you would like to add does not show up in the list, you need to install the tool first.

Click the Create Tool Link button.

Select a tool from the drop-down list.

Enter the site information and save.

  1. The Site ID is required.
  2. The Tool Title is also required.
  3. (Optional) Enter a Description for the tool.
  4. (Optional) Specify the Frame Height for the tool.
  5. (Optional) Check the box to Protect Launches.
  6. (Optional) Specify Custom Parameters if desired.
  7. Check the box to Add Site Link. This will place a link to the tool in the destination site's Tool Menu (i.e. left navigation menu).
  8. Click Save to save your settings.

View tool links to sites.

Once you have added the tool link, you will see it in the list of tool links to sites in the system. One tool may have many links if it has been added to several sites.
