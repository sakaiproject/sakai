How do I deploy a Stealthed LTI tool to a site or sites?

Once the system administrator has installed a stealthed (i.e. hidden or restricted access) external tool in the system, the administrator may deploy it to one or more sites. Non-stealthed LTI tools do not need to be deployed to sites, since they appear in the list of available tools by default.

Note: When stealthed tools are deployed to multiple sites, they are not automatically added to the site. Instead, they become available in the list of tools for instructors to add to the site via Lessons, Assignments, Rich Text Editor, and/or Site Info > Manage Tools (depending upon the tool placement options selected by the admin). Ordinarily, stealthed tools are not shown in the list of tools available for selection.

Go to External Tools.

Select the External Tools link from the Tool Menu in the Administration Workspace.

Click the number link in the Deployment column for the tool you'd like to add.

Note: Only previously installed tools will appear in this list. If the LTI tool you would like to add does not show up in the list, you need to install the tool first.

The list of sites where the tool is currently deployed will be shown.

To deploy the tool to additional sites, select the Deploy Tool to Sites button.

Enter the Site ID/s and Save.

  1. Enter one or more site IDs into the field provided. If you enter multiple site IDs, they should be on separate lines without any punctuation.
  2. Optionally, you can include notes that will be applied to all of the deployed sites.
  3. Click Save.

View deployed sites list.

Once you have deployed the tool to the specifies sites, you will see the updated list of all the sites currently in the system with the tool currently deployed.

Click Done to return to the External Tools main page.

The tool Deployment number will be updated.

Notice that the number of deployments in the tool table will be updated to include the sites where you have just deployed the tool.

To add the tool to the site menu, go to the site and select the tool from Site Info > Manage Tools.

Remember that adding the stealthed tool does not add it to the left navigation menu. The admin or the owner of the site will need to go to the site and select the tool in Site Info > Manage Tools to add it to the menu.
