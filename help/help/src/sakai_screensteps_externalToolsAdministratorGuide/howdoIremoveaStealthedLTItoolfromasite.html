How do I remove a Stealthed LTI tool from a site?

When a stealthed tool has been manually deployed to a site, and a site link to the tool has been created within that site, the instructor cannot remove the tool via Site Info > Manage Tools. Instead, one of the following methods should be used.

Note: Removing the tool link will not prevent the instructor or admin from re-adding the stealthed tool to the site. If you wish to remove access to the stealthed tool completely, the admin should delete the stealthed tool deployment.

Removing the stealthed LTI tool link as an instructor.

Go to Site Info.

Select External Tools.

Select Tool Links.

Select the Delete link in the Action column for the stealthed tool link you would like to remove.

Select Delete again to confirm removal.

View confirmation.

A success message will display and the tool link will no longer appear in the site Tool Menu.

Removing the stealthed LTI tool link as an admin.

In Administration Workspace, go to External Tools.

Select Tool Links.

Locate the tool link for the stealthed tool and the site link you would like to remove.

Tip: You may search by tool title, url, site title, added on, site owner, site contact, or department to help you locate the tool.

Select the Delete link in the Action column for the stealthed tool link you would like to remove.

Select Delete again to confirm removal.

View confirmation.

A success message will display and the tool link will no longer appear in the site's Tool Menu.

Deleting the stealthed tool deployment for a site.

In the Administration Workspace, go to External Tools.

Select the number link the Deployment column for the stealthed tool you would like to remove from a site.

You will see a list of sites to which the selected tool has been deployed.

Select the Delete link in the Action column for the site deployment you would like to delete.

Select Delete again to confirm deletion.

View confirmation.

A success message will appear, and the site will no longer appear in the list of deployed sites for this tool.

Note: When the Manual Tool Deployment page is used to remove a tool, but the navigation link (i.e. site tool link) remains, launching the tool displays the message “External tool item is missing or improperly configured.” This is expected behavior, since the tool is no longer available within the site.
