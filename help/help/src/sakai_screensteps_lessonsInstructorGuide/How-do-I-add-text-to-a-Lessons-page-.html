How do I add text to a Lessons page?

Instructors can add a text box to any point in the Lesson Page.

Go to Lessons.

Click on the Lessons Page Title (e.g. Unit #1) in the Tool Menu to display the page.

Note: The default or blank Lessons page contains text and images that explain the basic functionality of the Lessons tool.

Click Add Content, then Add Text.

From the Add Content drop-down menu, select Add Text.

This will display the Lessons Page Rich Text Editor text box.

Enter your text content.

Use the Rich Text Editor tools to format the text. When finished, click Save.

After saving, you will return to the Lessons page with the new text displayed.

Note: The Lessons tool descriptive text disappears from the page after content has been added.

Tip: Text boxes can be added at any point in the Lessons Page. You may want to intersperse text items with other types of Lessons content, such as images, embedded video, assignment or assessment links, etc.
