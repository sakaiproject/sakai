How do I create a direct link to a tool?

Use Link to create a direct link to a specific tool within your site. Users which click the direct link will jump straight to the tool without the need to use the navigation. Users can only use direct links if they have permission to view the tool.

Select the Link button for the tool you would like to link to directly.

While viewing a tool to which you would like to generate a direct link, click Link near the header of the page.

Copy the direct link.

Select and copy the information in the direct URL field.

Or, copy the short URL.

Generate a short URL for sharing.

  1. Select Short URL.
  2. Select and copy the information in the direct URL field.

Share the URL with site participants.

Use tools within the site like Announcements or Overview to share the URL with students, or share the URL or email.
