How do I use Help from the site header?

Open the Help area to view help information about the tool or feature you're currently using in the system. As an example: while using the Assignments tool, click Help to view help information about the Assignments tool.

Locate Help while using the system

While using any tool in the system, click Help (View contextualized help for this page) in the site header area of the page to access help information about the tool.

Opening Help while using a tool

Click Help in the top navigation to view Help for the currently selected tool.

While using a tool, click Help (View contextualized help for this page) in the site header area of the page.

View help information.

  1. View help information in the content area of the page.
  2. Or, enter information in the Search Help field to locate other help articles.
