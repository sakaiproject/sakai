How do I search the system?

Use Search to locate content from all your sites based on search terms. Search will only fully display content to which you have access.

Click Search.

Click the search icon (Search for Content in Sakai) in the site header area of the page.

Enter a search term.

  1. In the text field, enter a search term.
  2. Select the Search button to run the search.

Click a result to jump to the content.

Click a result to jump to the content's location within a site.
