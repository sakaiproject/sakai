How do I add a website link to a Lessons page?

Instructors can add links to Internet websites on a Lessons page.

Go to Lessons.

Click on the Lessons Page Title (e.g. Lesson 4) in the Tool Menu to display the page.

Note: The default or blank Lessons page contains text and images that explain the basic functionality of the Lessons tool.

Click Add Content, then Add Content Link.

Zoom

From the Add Content drop-down menu, select Add Content Link.

Enter the web address.

Zoom

Type (or copy and paste) the web address for the website in the text box labeled Or add a URL.

Click Save.

Zoom

View link on the page.

Zoom

Saving returns the display to the Lessons page with the link to the website.

Click Edit. (Optional)

Zoom

Add a description, then click Update Item.

Zoom

View updated item on the page.

Zoom

Updating returns the display to the Lessons page with the link and the link description shown.
