How do I change the background color of a block?

Go to Lessons.

If you titled your page something other than Lessons, select the title of the page as it appears in your Tool Menu.

Click on the Lessons tool in the Tool Menu to display the page.

Note: The Lesson page must already contain content before you can change the color of the block.Â 

Select the column properties icon in the top left of the block.

Choose the desired color from the drop-down menu.

Note: To change the color of the block to be all one color, you will need to use the Legacy Colors. The other colors will change the color of the borders.

Click Save.

The selected color will display.
