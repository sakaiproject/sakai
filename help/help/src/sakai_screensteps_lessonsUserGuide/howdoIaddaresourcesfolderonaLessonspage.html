How do I add a resources folder on a Lessons page?

Instructors can add a link to an existing Resources folder on a Lessons page. This will display a listing of the folder contents on the page.Â 

Go to Lessons.

If you titled your page something other than Lessons, select the title of the page as it appears in your Tool Menu.

Click on the Lessons tool in the Tool Menu to display the page.

Note: The default or blank Lessons page contains text and images that explain the basic functionality of the Lessons tool.

Click Add Content, then select Add Resources Folder.

From the Add Content drop-down menu, select Add Resources Folder.

Select the Resources folder.

  1. Browse the directory listing of your existing Resources folders. Once you have located the folder you want to link on the Lessons page, click on the folder name to select it.
  2. Notice that the folder path appears in the Selected Folder field.
  3. Click Save to save your selection.

The folder contents will display on the page.
