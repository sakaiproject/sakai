What information is displayed on the Site Dashboard?

The Dashboard in a course or project site shows information from the current site only.

Go to the Dashboard tool.

Select Dashboard from the Tool Menu in your course or project site.

The Dashboard in a course or project site contains the following:

  1. Click Edit Dashboard to customize the layout of the page. In course or project sites, this option is available to instructors and site owners only.
  2. The course overview area displays information about the site. Instructors and site owners can edit the course overview.
  3. The Calendar widget displays calendar events, such as deadlines or assessments, from the current site.
  4. The Announcements widget displays recent announcements from the current site.
  5. The Discussions widget displays new Messages and Discussions from the current site.
  6. The Grades widget displays grade information from the current site. (This widget has a slightly different view for instructors than for students.)

Note: Edit Dashboard is available to instructors and site owners only when used in a course or project site.
