What information is displayed on the Home Dashboard?

The Dashboard in your Home site displays information from all of your current sites.

Go to the Dashboard tool.

Select Dashboard from the Tool Menu in your Home site.

The Dashboard in your Home site contains the following:

  1. Click Edit Dashboard to customize the layout of the page. This option is available to all users in the system.
  2. The Tasks widget displays deadlines and other calendar events from all of your current sites. Click Add a new task to add personal tasks the widget.
  3. The Announcements widget displays recent announcements from all of your current sites.
  4. The Discussions widget displays new Messages and Discussions in all of your current sites.
  5. The Grades widget displays grade information and grading tasks across all your current sites. This widget is intended primarily for instructors.
  6. The Calendar widget displays calendar events, such as deadlines or assessments, from all your current sites.

Note: Edit Dashboard is available to all users when used with the Home site's Dashboard tool.
