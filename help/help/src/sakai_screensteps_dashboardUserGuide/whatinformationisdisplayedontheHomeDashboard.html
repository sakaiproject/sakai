What information is displayed on the Home Dashboard?

The Dashboard in your Home site displays information from all of your current sites.

Go to the Dashboard tool.

Select Dashboard from the Tool Menu in your Home site.

The Dashboard in your Home site contains the following:

  1. Click Edit Dashboard to customize the layout of the page. This option is available to all users in the system.
  2. You can use the filter and sorting drop down menus to filter your Dashboard by type of pinned site, term, or to change the sort order.
  3. The Course Cards widget displays a card for each of your enrolled and pinned sites. You can navigate to a course by clicking on the course card. You may see icons on these cards if there is new activity available.
  4. The Announcements widget displays recent announcements from all of your current sites.
  5. The Calendar widget displays calendar events, such as deadlines or assessments, from all your current sites.
  6. The Grades widget displays grade information from all your current sites. (This widget has a slightly different view for instructors than for students.)
  7. The Discussions widget displays new Messages and Discussions in all of your current sites.

Note: Edit Dashboard is available to all users when used with the Home site's Dashboard tool.
