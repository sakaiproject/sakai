How do I edit the Site Dashboard?

Only instructors and site owners can edit the Dashboard in course and project sites. All site participants will see the same site Dashboard settings as those defined by the instructor or site owner.

Go to the Dashboard tool.

Select Dashboard from the Tool Menu in your course or project site.

Click Edit Dashboard.

Edit the Dashboard layout.

Click Layout to open dashboard layout options menu.

Select one of the layout options and then click Select.

Option 2 contains the course image. Editing this image will also set the background image that is shown in the Home Dashboard Course Cards widget for this site.

Edit the course image.

To update the course image, click Edit image.

  1. Click Choose File to upload a new image from your device.
  2. Click Zoom image in or Zoom image out to zoom in or out on a section of the image.
  3. Click Pan image up or Pan image down to move the image center up or down within the frame.
  4. Click Pan image left or Pan image right to move the image center left or right within the frame.
  5. Click Rotate image to rotate the image by 90% within the frame.
  6. Click Done.

Edit the course overview.

Enter updates to the course overview in the Rich Text Editor.

Remove widgets.

If there are any widgets on the page that you would like to remove, click the X [Remove] icon to remove the widget.

Add widgets.

If you would like to add a widget to the page:

  1. Click Add a Widget.
  2. Select a widget to add. In the example, the Calendar widget is available.

Note: In many sites, all available widgets are added by default. If you attempt to add a widget when they are all currently enabled, you will not see any widgets in the list of items to add.

Move widgets.

Use the arrows (Move up, Move down, Move left, Move right) to move widgets to a different location on the dashboard.

Click Save once you have completed your edits.
