How do I edit the Home Dashboard?

All users can customize their own personal Dashboard in their Home site.

Go to the Dashboard tool.

Select Dashboard from the Tool Menu in your Home site.

Click Edit Dashboard.

Remove widgets.

If there are anywidgets you don't want to see on your Dashboard, click the X [Remove] icon to remove the widget/s.

Add widgets.

  1. Click Add a Widget.
  2. Select a widget to add. In the example, the Announcements widget is available.

Note: In many dashboards, all available widgets are added by default. If you attempt to add a widget when all are currently enabled, you will not see any items in the list to be added.

Move widgets.

Use the icons (Move up, Move down, Move left, Move right) to move widgets to a different location on the dashboard.

Click Save once you have completed your edits.
