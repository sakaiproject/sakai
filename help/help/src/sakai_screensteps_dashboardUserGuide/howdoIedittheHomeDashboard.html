How do I edit the Home Dashboard?

All users can customize the Dashboard in their Home site.

Go to the Dashboard tool.

Select Dashboard from the Tool Menu in your Home site.

Click Edit Dashboard.

Remove widgets.

Click the Remove icon to remove the widget.

Add widgets.

  1. Click Add a Widget.
  2. Select a widget to add. In the example, the Tasks widget is available.

Note: In many dashboards, all available widgets are added by default.

Move widgets.

Use the icons (Move up, Move down, Move left, Move right) to move widgets to a different location on the dashboard.

Click Save once you have completed your edits.
