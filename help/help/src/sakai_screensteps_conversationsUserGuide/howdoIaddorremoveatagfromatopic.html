How do I add or remove a tag from a topic?

Tags are used to organize related topics. By default, instructors in the course can create tags, while any site participant can apply a tag to their new or existing topics. You can tag entire topics but not individual posts within topics.

If you haven't yet set up tags for your course, see How do I manage tags for my course in Conversations?.

Go to the Conversations tool.

Select the Conversations tool from the Tool Menu of your site. 

Adding a tag to a new topic

Click Create new topic.

See How do I post a topic in Conversations? for the complete steps to posting a new topic.

Author the topic.

Use the Tag topic menu.

  1. Open the Tag topic menu.
  2. Select a tag.
  3. Click Add.

Note: If there are no tags available for the site, or you want to add a new tag, use the Edit tags for this course button to create tags. You must have the "Setup new tags" permission in order to do this. If you do not have this permission, you will not see the button.

The added tag will display.

Publish the topic.

Click Publish.

Adding a tag to an existing topic

Click on a topic.

Click on an existing post in the topic list.

Click Edit.

  1. Click the Topic options menu icon in the topic area.
  2. Click Edit.

Use the Tag topic menu.

  1. Open the Tag topic menu.
  2. Select a tag.
  3. Click Add.

Note: If there are no tags available for the site, use the Edit tags for this course button to create them. You must have the "Setup new tags" permission in order to do this.

The added tag will display.

Publish the topic.

Click Publish.

Removing a tag

Click on a topic.

Click on an existing post in the topic list.

Click Edit.

  1. Click the Topic options menu icon in the topic area.
  2. Click Edit.

Click on the X next to the tag to remove it.

Click the remove icon.

Publish the topic.

Click Publish.
