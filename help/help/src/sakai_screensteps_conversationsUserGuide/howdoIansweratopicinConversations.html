How do I answer a topic in Conversations?

Go to the Conversations tool.

Select the Conversations tool from the Tool Menu of your site. 

Click on a topic.

Click on a topic in the topic list. Question topics display a question icon (This is a question topic) near the title of the topic, while Discussion topics display a speech bubble icon (This is a discussion topic).

Click Answer this question.

In the topic area, click Answer this question.

Enter an answer.

  1. Enter an answer in the Answer this question field.
  2. Optionally activate Private to make the answer visible only to instructors and the original poster of the topic.
  3. Optionally activate Post anonymously. This option is available only if the original poster activated it when creating the post.
  4. Click Publish.

The answer will display within the topic.

Note: When the Instructor posts an answer to a question, the topic is marked as "Answered" and a green check mark displays for that topic. This is only true of instructor posts.
