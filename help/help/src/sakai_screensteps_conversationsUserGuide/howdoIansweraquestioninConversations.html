How do I answer a question in Conversations?

To post an answer to a question, first select the Conversations tool in the left-hand menu.

Select the question you'd like to answer from the list of topics on the left. 

If the question has not yet been answered or to provide an additional answer, select Answer this question. 

Enter the answer in the text box that appears. If you would like only the person who posted the question (and the instructor) to see your response, you can check the box next to Private below the text entry box.

If the author of the question enabled anonymous responses, you will see an option to Post anonymously below the text entry box as well.

Select Save as Draft if you would like to revise your answer further before posting. Your draft will then be displayed under the question with a "draft" notation. 

Or, if you're ready to post the answer immediately, select Publish.
