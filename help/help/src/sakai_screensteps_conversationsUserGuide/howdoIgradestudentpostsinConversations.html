How do I grade student posts in Conversations?

You must have grading permissions in Conversations in order to see the grading options described below.

Go to the Conversations tool.

Select the Conversations tool from the Tool Menu of your site. 

Select a topic.

Click on the topic that you would like to grade in the topic list.

Select Grade for the student post to be graded.

Grade the post.

  1. Enter a score for the post.
  2. Optionally, enter grader comments about the post. (Note that comments can only be entered as plain text.)
  3. Click Submit Grade to send the grade and comments to the Gradebook.
