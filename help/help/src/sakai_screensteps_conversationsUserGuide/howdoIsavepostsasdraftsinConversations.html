How do I save posts as drafts in Conversations?

Use Save as Draft to save progress on a post that you are not yet ready to publish to other site participants.

Go to the Conversations tool.

Select the Conversations tool from the Tool Menu of your site. 

Click Create new topic.

Author the topic.

See How do I post a topic in Conversations? for the complete steps to posting a new topic.

Click Save as Draft.

Return to the draft.

Topics that you have saved as drafts will appear in the DRAFT area of the topic list. Draft topics will also display the message "This topic is a draft" above the post when you select it for viewing. Draft items are only visible to the author. Click on a draft topic in the topic list to continue editing and/or publish it.

Click Edit.

  1. Click the Topic options menu icon in the topic area.
  2. Click Edit.
