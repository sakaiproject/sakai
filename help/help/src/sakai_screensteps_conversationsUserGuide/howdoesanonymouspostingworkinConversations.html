How does anonymous posting work in Conversations?

When you select Create new topic to add a question to Conversations, there are two different Anonymous options available. 

  1. Anonymous - Hide name on post: Select this option if you would like your question topic to show Anonymous instead of the author’s name.
  2. Anonymous Comments - Allow anonymous comments: Select this option if you would like to allow users to post answers and replies/comments on answers within this question topic with Anonymous shown as the author.

Note: The instructor or site owner will always see the name of the topic or post author, regardless of the anonymous setting.
