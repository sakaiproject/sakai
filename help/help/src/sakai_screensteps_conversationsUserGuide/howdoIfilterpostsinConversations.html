How do I filter posts in Conversations?

Go to the Conversations tool.

Select the Conversations tool from the Tool Menu of your site. 

Filtering with the Filter menu

Open the Filter menu.

Select a filter.

Select a filter from the list.

  • Filter: questions: filter for topics that use the question type.
  • Filter: answered questions: filter for topics that use the question type and that have an accepted answer.
  • Filter: discussions: filter for topics that use the discussion type.
  • Filter: discussions with posts: filter for topics that use the discussion type and that have responses.
  • Filter: bookmarked: filter for topics that you have bookmarked.
  • Filter: moderated: filter for topics that are moderated.
  • Filter: unviewed: filter for topics that you have not yet viewed.

Filtering with the Tag menu

Open the Tag menu.

Select a tag.

Select a tag to display topics which possess the tag.

Note: The Tag menu displays tags assigned to topics by their authors. Course instructors must configure tags before site participants can add them to topics.
