How do I view reports for a specific site?

Go to Sitestats Admin.

Select the Sitestats Admin tool from the Tool Menu in the Administration Workspace.

Locate the site you want to view.

Zoom

Enter a search term and click Search to locate the site you want to view.  

All sites matching your criteria will be displayed.

Click on the title of the site.

Zoom

The site's Statistics will display.

Zoom

From this view, you may run any reports within the site the same way an instructor/maintain user would run reports within an individual site. See What is the Statistics tool? for more information on how to run reports.
