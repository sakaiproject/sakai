How do I create a New Account in Sakai?

If your instance of Sakai allows users to create their own accounts, you will see a New Account tool in the Gateway area.

Note: Many institutions control user account creation via integration with their registration system, so this option may not be available in many instances.

From the Gateway page, select the New Account tool in the Tool Menu.

Enter your account information in the fields provided.

Keep in mind that you will need to create a unique User id and a Password that meets the security criteria (number of characters, upper/lowercase, etc.) set at the system level.

After entering your information, select Create Account to create your new account.

Your new account will be created and you will be logged in as that user.
