How do I reset my password in Sakai?

If your instance of Sakai allows users to reset their passwords, you will see a Reset Password tool in the Gateway area.

Note: Many institutions control user account credentials via integration with their registration system, so this option may not be available in many instances.

From the Gateway page, select the Reset Password tool in the Tool Menu.

Enter your email address and select the Send Password button.

If the email address you entered is associated with a user account in the system, you will be sent an email with instructions on resetting your password.
