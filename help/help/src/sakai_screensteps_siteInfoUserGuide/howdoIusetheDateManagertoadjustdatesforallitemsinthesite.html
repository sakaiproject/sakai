How do I use the Date Manager to adjust dates for all items in the site?

Note: To complete these steps, you must be assigned a role having the necessary permissions within the site.

Go to Site Info.

Select the Site Info tool from the Tool Menu of your site.

Click on Date Manager.

Select the desired tool/s to expand each section and view items within that tool.

Note: Different tools may have different date options available depending upon the individual item settings and capabilities of the tool.

Select the calendar icon next to each date field/s you would like to modify and enter a new date.

Scroll down and select the Save button to save all of the date changes on the page.

Select the Save Changes button to confirm the changes to items listed in the confirmation dialog.

A message will display at the top of the page indicating that the dates were updated.
