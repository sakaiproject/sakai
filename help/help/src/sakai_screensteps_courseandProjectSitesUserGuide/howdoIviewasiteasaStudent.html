How do I view a site as a Student?

Use Student View to experience your course site as a different user type. This can help you to spot opportunities to enhance your site or possible points of confusion for site participants.

Select Enter Student View.

In the top of the content area, just below the global navigation icons, select the Enter Student View link.

Or, select the View Site As drop down menu.

If your institution has configured role swap feature to work for additional roles, such as TAs, you will see a drop down menu instead of the Enter Student View link. From the drop down menu, select a user type, such as Teaching Assistant or Student as shown in the example image.

Note: different user types may be displayed in your institution's installation of the system.

Navigate the site as a different user type.

Navigate the site as the user type that you selected from the View Site As menu. The user type will display to you near the top of page in the breadcrumb area.

Also, notice that the browser window has a purple border. This border indicates that you are viewing the site as a user type different from your own.

Return to your normal view.

Click Exit Student View to leave the student view and return to your normal view.
