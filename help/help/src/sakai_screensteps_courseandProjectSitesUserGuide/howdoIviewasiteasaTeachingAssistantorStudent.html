How do I view a site as a Teaching Assistant or Student?

Use View Site As to experience your course site as a different user type. This can help you to spot opportunities to enhance your site or possible points of confusion for site participants.

Open View Site As.

In near the top of page in the breadcrumb area, open View Site As.

Choose a user type.

Select a user type, such as Teaching Assistant or Student as shown in the example image.

Note: different user types may be displayed in your institution's installation of the system.

Navigate the site as a different user type.

Navigate the site as the user type that you selected from the View Site As menu. The user type will display to you near the top of page in the breadcrumb area.

Return to your normal view.

Click the exit button to return to your normal view. As shown in the example image, click Exit Student View to leave the student view and return to the normal view.
