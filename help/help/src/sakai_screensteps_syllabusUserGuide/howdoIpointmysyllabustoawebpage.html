How do I point my syllabus to a webpage?

Go to Syllabus.

Select the Syllabus tool from the Tool Menu in your site.

Select Redirect.

Enter the URL of the webpage location of your syllabus

Click Save

If you would like to clear the URL, select the X icon within the field to remove it.

When you return to the Syllabus tab, you will see the URL redirect displayed there.

The redirect URL will open in a new window.
