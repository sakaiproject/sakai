How do I add my syllabus as a file attachment?

Go to Syllabus.

Select the Syllabus tool from the Tool Menu in your site.

Click Add.

Give your file attachment a title and select Create a single syllabus item, then click Add.

The Edit Syllabus Item screen will display. Select the Add Attachments button on this screen.

Click the Add attachments link to browse for your file. Optionally, you may also change the title of your Syllabus item, or add text within the Rich Text Editor which will appear above the file attachment.

Select your syllabus file.

You may choose to 1) Upload  a file, 2) Link to a website, or 3) Select a file from Resources.

Upload a local file

Select Choose File to browse for the file on your local computer.

Click Continue to attach the selected file.

Your file attachment will display.

Link to a website.

Select the Link to website tab.

Enter the URL.

Click Continue to attach the website link.

Your website link will display. Select Publish or Save Draft to save your change.

Select from Resources.

Select the Select from Resources tab.

The site Resources will display. Select the Attach a Copy link for the item you want to attach.

Click Continue to attach the file.

Your file attachment will display. Select Publish or Save Draft to save your change.

When you return to the Syllabus tab, you will see the syllabus item and file attachment displayed.
