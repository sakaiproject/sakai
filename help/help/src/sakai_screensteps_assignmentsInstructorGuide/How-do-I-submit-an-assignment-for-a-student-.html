How do I submit an assignment for a student?

Select the Assignments tool from the Tool Menu in your site

Ensure you are on the Assignment List (if not click the Assignment List button)

Go to the Assignment tool.

Select the Assignments tool from the Tools Menu in your site.

Use the View dropdown to select Assignment List by Student option.

Find the student either by group or search from the list.

Zoom

Once the student is selected the set assignments will be revealed

For the student you wish to submit for click Submit as Student

Zoom

Follow the prompts to submit
