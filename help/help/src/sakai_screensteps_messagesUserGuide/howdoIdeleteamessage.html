How do I delete a message?

A message can be deleted if it is no longer needed.

Go to Messages.

Select the Messages tool from the Tool Menu in your site.

Open the folder containing the message that you wish to delete.

Select the message.

Select the checkbox associated with the message that you wish to move.

Click Delete.

View the confirmation message.

Information will display at the top of the page to confirm that you've deleted the message.

Note: Deleted messages are moved to the Deleted folder.
