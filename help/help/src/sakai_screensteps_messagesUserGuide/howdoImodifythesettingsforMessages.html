How do I modify the settings for Messages?

This feature allows users to modify the default Messages tool settings. Users with site owner permission have additional options for configuring the tool within the site.

Go to Messages.

Select the Messages too from the Tool Menu in your site.

Click Settings.

Modify the site owner (Instructor) Settings options:

In the Site Settings area, instructors or site owners can choose:

  1. whether or not site participants can forward messages to recipient email addresses
  2. which groups (if any) are hidden in the message recipient list

In the Personal Settings area, you can optionally set a personal forwarding email address for your own messages.

Modify the site participant (Student) Settings options:

If the site owner has allowed sending copies (i.e., forwarding messages), then site participants can use the Email address for forwarding field to optionally specify an address.

Click Save Settings.

Click Save Settings to save your changes.
