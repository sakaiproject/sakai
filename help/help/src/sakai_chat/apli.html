The following is an overview of the Chat Room feature.

What it does

Chat is a tool for real-time, unstructured conversations with users who are signed on to the site at the same time.

Instructors can easily create an "Online Office Hours" Chat Room for student questions and answers. Dispersed collaboration groups can use the room as a space to have conversations across distances or catch up with conversations they may have missed.

Key concepts

The Chat tool allows for more than one room that an instructor or site owner can create for specific kinds of chats. These additional chat rooms can be created using the Options feature.

Changes that an instructor makes to the Chat Room display also apply to how the Chat Room features appear to students:

  • Hide Date button:Click theHide Datebutton to hide the date next to each respondent's name. Click again to show the date.



  • Hide Time button:Click theHide Timebutton to hide the time. Click again to show the time.



  • Options button:This button includes options to choose how many messages will be displayed.

Things to consider

  • The Chat Room tool alerts users on the Chat Room page to the other participants who are also viewing that page. This way, users know who is available to talk.



  • By default, Chat messages are saved and visible to all users so that all site participants can benefit from clarifying conversations and questions and answers. By default, all Chat messages are shown for at least three days.
