To add an attachment to an activity on your Schedule, follow these steps:

  1. In the menubar, click Schedule, then click New... .

  2. Type a title for the activity.

    Note: A red dot means that that information is required.

  3. Type a description for the new activity.

  4. To choose the date and start time for the activity, click the arrows.

  5. Click the Set frequency... button to select the frequency. Then enter information about how often the activity will occur and when it will end.

    Note: If you do not choose a frequency, your activity will occur once.

  6. Click the arrows to select the duration of the activity.

  7. To choose which type of activity will occur, pull down the Item type menu and select an activity type. It is optional to add a location for your activity.

  8. Click the Attachments button on the window for creating a new activity.

  9. In the Attachment window, you can choose to:

    • Attach a local file from your computer:

      Click Local File... , then type the filename in the box provided, or browse for the file, select it, and click Attach. In the Attachments window that appears, click Save

      Note: When uploading or attaching a file, you must include the file's extension (e.g., .gif, .doc, .html, or .jpg). If you do not, anyone who attempts to display or download the file will get an error message.

    • Attach a website URL:

      Click Website URL... , then type the URL in the box provided. You do not need to include http://. Then click Attach. In the Attachments window that appears, click Save.

    • Attach an item from the resources already in this course website:

      Click From Resources... , then choose the resource you would like to add by clicking the box next to it, and click Done. In the Attachments window that appears, click Save.

Note: When you have finished adding an attachment, remember to add your activity to the Schedule.
