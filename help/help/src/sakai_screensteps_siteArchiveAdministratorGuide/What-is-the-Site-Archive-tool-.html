What is the Site Archive tool?

The Site Archive tool allows admin users to "back up" or archive sites within the system, as well as import sites from existing archives. The archives created via Site Archive may be kept for institutional records, or used for transferring a site from one Sakai instance to another.

Note: The archive file contains all site content, but does NOT contain user data such as student activity, messages, assignment submissions, forum posts, etc.

To access this tool, go to Site Archive from the Tool Menu in the Administration Workspace.
