Changing a user's name

Note: To use this tool, you must have administrator rights.

To change a user's name, follow these steps:

  1. From your My Workspace, click the Users tab. This will display a list of users defined within the system.

  2. Under "User Id", click the username of the user you want to edit.

  3. Edit the "First Name:" and/or "Last Name:" field(s) as you wish, and then click Save.
