Depending on the kind of information you want to display in the Worksite Information window, follow the appropriate instructions below:

A web page

To make a web page appear in the Worksite Information window, follow these steps:

  1. Select a course or project site.

  2. If the site's homepage is not already open, click the Home button in the menubar.

  3. In the Worksite Information window, click the Options... button.

  4. Click the radio button next to Display this URL.

  5. In the "URL:" field, type the URL (you may omit the http:// prefix).

  6. Click Save.

You may need to refresh your browser to view your new Worksite Information.

A text description

To add a text description, follow these steps:

  1. Select the course or project site.

  2. If the site's homepage is not already open, click the Home button in the menubar.

  3. In the Worksite Information window, click the Options... button.

  4. Click the radio button next to Display this worksite's description (from setup) .

  5. Click Save. The text you typed in the "Description" box when you created the site will appear. You may need to refresh your browser to view your content.

If you left the "Description" box empty when you created the site, you can add a description with the following steps:

  1. Click My Workspace.

  2. Click Worksite Setup.

  3. Click the box next to the course site.

  4. Click Revise.

  5. Click Edit Site Info... .

  6. Type a description in the box provided.

  7. Click Continue, and then Finish.

  8. Open the site again. The description should now appear in the Worksite Information window. You may need to refresh your browser to view it.

An HTML file of your own

To display an HTML file you have created and uploaded to the "Resources" area, follow these steps:

  1. Click Resources to view the Resources list.

  2. Click the name of your HTML file. It will display in a new browser window.

  3. In the new window, highlight the HTML file's URL. Using your browser's Copy command, copy the URL.

  4. Return to your Resources window. Click the Home button at the top of the menubar to go to the course or project site's home page.

  5. In the Worksite Information window, click Options... .

  6. Click Display this URL.

  7. Paste your URL into the "URL:" field.

  8. Click Save.

  9. View your HTML file in the Worksite Information window. You may need to refresh your browser.
