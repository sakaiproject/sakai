How do I add a chat room?

Go to Chat Room.

Select the Chat Room tool from the Tool Menu of your site.

Click Add Room.

Enter the room details.

  1. Enter a title for the chat room in the Title field.
  2. Enter a short description of the room in the Description field that will display to site participants when they view a list of available chat rooms in the site. (Optional)
  3. Use the Recent Chat Display are to select the number of messages you would like to be displayed in the message history.
  4. Activate Allow chat participants to change the chat display settings for their own chat window to allow site participants to customize their own experience. (Optional)
  5. Set a start and end date for the chat room in the Allowed Posting Dates area. Leaving the dates blank will allow the room to always remain open. (Optional)
  6. Click Update Options.

The new chat room will display in the list.
