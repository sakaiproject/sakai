How do I change to a different chat room?

Go to Chat Room.

Select the Chat Room tool from the Tool Menu of your site.

As an instructor:

Click Options.

Select a chat room.

Click the title of a chat room to enter the chat room.

Tip: use Set as Default to make a chat room the default location for students when they visit the Chat Room tool.

As a student:

Click Change Room.

Select a chat room.

Click the title of a chat room to enter the chat room.
