How do I change the Chat Room tool permissions?

Go to Chat Room.

Select the Chat Room tool from the Tool Menu of your site.

Click Permissions.

Modify the permissions for the roles listed.

  1. Check the box next to each permission you would like to allow for a given role. Un-checking a box will remove the corresponding permission.
  2. Click Save.

Note: You may see different roles listed depending on the roles that exist in your site. A role must be present in the site in order for you to modify its permissions.
