How do I read, post, or delete Chat Room messages?

Go to Chat Room.

Select the Chat Room tool from the Tool Menu of your site.

Read and post messages:

When you open the Chat Room tool, you will enter the default room (as specified by the site leader).

  1. Existing messages appear in chronological order, with the most recent at the bottom. Messages will always identify the sender
  2. Enter a new message in the Enter chat message field.
  3. Click Add message to send the new message.

Note: No one can see your message until you click Add message or use Enter/Return on your keyboard. Once you've posted a message, only an instructor or a participant with special permission can delete your message.

Delete messages. (Optional)

To delete an existing chat message that you or other users have entered, click the trash can icon (Delete this Message) next to the message.

Note: If you don't see a trash can icon, you don't have permission within the Chat Room tool to delete the message.

Confirm deletion.

You are be prompted to confirm the deletion. If you are sure you want to delete the message, click Delete.
