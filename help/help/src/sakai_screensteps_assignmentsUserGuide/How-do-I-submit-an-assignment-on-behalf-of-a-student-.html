How do I submit an assignment on behalf of a student?

Go to Assignments.

Select the Assignments tool from the Tool Menu in your site.

From the View drop-down menu, select Assignment List by Student.

Zoom

Click the Grade link for the assignment you would like to grade.

Select a student.

Zoom

You will see a list of all the students in the class. Click on the triangle icon to the left of the student's name to expand and view submission information for that student.

Click Submit on behalf of Student.

Zoom

Submit the student assignment.

Zoom
  1. Enter an in-line submission (if applicable).
  2. Select a file to attach (if applicable).
  3. Click Submit to submit the student assignment.

A submission confirmation will display.

Zoom

When you view the list, you will see the name of the instructor next to the submission.
