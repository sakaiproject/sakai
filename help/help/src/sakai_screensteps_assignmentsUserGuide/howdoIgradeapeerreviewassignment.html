How do I grade a peer review assignment?

Go to Assignments.

Select the Assignments tool from the Tool Menu in your site.

Select the peer review assignment to grade.

Click the Grade option associated with a peer review assignment.

Review the peer assessment scores.

The scores provided by peer reviewers display to you on this page. If more than one review was assigned for each student, the reviewer grade will be an average of all the reviewer grades submitted.

By default, the grade for the assignment is set to the reviewer grade, or the average of the reviewer grades. You can override this grade if you choose.

Click on the reviewer name to review their feedback.

Review the individual reviewer's feedback.

  1. The score that the reviewer has awarded this submission displays to you in the Grade section.
  2. The reviewer's feedback displays to you in the Review Comments area.

Tip: If a reviewer has provided incorrect or inappropriate feedback, click Remove Review to delete the peer feedback and score.

Release grades.

To make the peer review scores the official grades for each student's assignment, return to the Submissions page and click Release Grades. This will release grades and feedback to students.

Or, enter a score adjustment and instructor comments.

Click on a student name to go to the detailed view of the student submission.

Adjust the score or enter comments.

Use the Assignment Submission area to update the grade and add instructor feedback:

  1. Enter a score adjustment in the Grade field.
  2. Add instructor comments using Feedback Comment.
  3. Attach a file with additional feedback with Add Attachments.
  4. Click Save to adjust the score and release it to the student later, or click Save and Release to Student to release the grade and feedback immediately.
