How do I grade a peer review assignment?

Go to Assignments.

Select the Assignments tool from the Tool Menu in your site.

Select the peer review assignment to grade.

Click the Grade option associated with a peer review assignment.

Note: If you select the Grade link before the peer evaluation window has closed, you will see a message indicating that Instructor grading is disabled until after the review period.

Review the peer assessment scores.

The scores provided by peer reviewers display to you on this page. If more than one review was assigned for each student, the reviewer grade will be an average of all the reviewer grades submitted.

If there was a grading rubric associated with this assignment, you will see individual rubric icons for each reviewer in the Reviewed By column. Selecting the individual reviewer icon will show the rubric selections made by that peer reviewer.

If you select the rubric icon in the Grade column, you will see an unscored rubric. If you choose to override the average review scores with an instructor score, this rubric will show the instructor's rubric selections.

By default, the grade for the assignment is set to the reviewer grade, or the average of the reviewer grades. You can override this grade if you choose.

Click on the reviewer name to review their feedback.

Review the individual reviewer's feedback.

  1. The student's submission (in the format specified in the assignment settings for the Type of submissions allowed) will appear at the top.
  2. If there was a grading rubric associated with this assignment, you will see the scored peer review rubric below the student's submission.
  3. The score that the reviewer has awarded this submission displays to you in the Grade section.
  4. The reviewer's feedback displays to you in the Review Comments area.
  5. If there is a reviewer attachment (optional) it will be shown in the Reviewer Attachments section.
  6. If a reviewer has provided incorrect or inappropriate feedback, you may select the Remove Review button to delete the peer feedback and score.
  7. Select Next to view the next reviewer's feedback, or Return to List to return to the list of student submissions.

Release grades.

To make the peer review scores the official grades for each student's assignment, return to the Submissions page and click Release Grades. This will release grades and feedback to students.

Or, enter a score adjustment and instructor comments.

Click on a student name to go to the detailed view of the student submission.

Adjust the score or enter comments.

Use the grading panel on the right to update the grade and add instructor feedback:

  1. Enter a score adjustment in the Grade field.
  2. Add instructor comments using Feedback Comment.
  3. Attach a file with additional feedback with Add Attachments.
  4. Click Save to adjust the score and release it to the student later (or click Save and Release to Student to release the grade and feedback immediately).
  5. The Peer Reviews will also display on the right, below the student's submission. You may click on a reviewer to see the reviewer's feedback.
