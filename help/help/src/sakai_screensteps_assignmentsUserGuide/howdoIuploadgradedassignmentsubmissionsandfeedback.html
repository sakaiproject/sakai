How do I upload graded assignment submissions and feedback?

Use Upload All to batch-upload grades and feedback that you've created outside of the site or offline. See How do I download assignments for grading offline? for information about downloading files for offline grading.

Create a zip file from the extracted folder on your computer.

If you have previously extracted the zip file on your computer, and then edited or added to the assignment grades or feedback, create a new archive or zip file for upload that includes your changes.

Create a zip file from a folder in Windows by right-clicking on the folder, and then selecting Send to and Compressed (zipped) folder. The zip file will have the same name as the folder you selected.

Tip: You can either compress/zip all existing assignment folders at once for upload, or just one assignment folder at a time.

Go to Assignments.

Select the Assignments tool from the Tool Menu of your site.

Click the Grade link for the assignment to be graded.

Click the Upload All link.

Select the archive file containing grades and feedback.

Click Choose File to browse for and select the archive file on your local computer..

Note: The archive file must contain a folder for the assignment and subfolders for each of the individual students. To ensure that your file is in the correct format, download the assignment submissions (or the template provided) directly from the Assignments tool in your site.

Select the desired upload and release options.

  1. Check All to select all of the upload options, or check individual items. The options available for upload are:
    • All
    • Student submission text (original student submitted text, possibly containing instructor added comments)
    • Student submission attachment(s)
    • Grade file (file at top level of archive)
    • Feedback text (the inline comments with student submission)
    • Feedback comments (comments.txt file if available in student's folder. Comments are put into the Instructor Comments field for each student's submission)
    • Feedback Attachment(s)
  2. Select Release uploaded information to students to release uploaded feedback to students immediately. Select Do not release uploaded information - I'll release it later to manually release grades to students at a later date.

Click Upload.

Click the Upload button to upload your file.
