How do I edit a post in Commons?

You must possess editing permissions in the Commons tool in order to edit a post.

Go to the Commons tool.

Select the Commons tool from the Tool Menu of your site.

Click Edit.

Locate the post that you would like to edit and click its associated Edit option.

Edit the post, and then click Post.

  1. Make changes to the content of the post.
  2. Click Post.

View the edited post.

View the edited post to verify that changes were made successfully.
