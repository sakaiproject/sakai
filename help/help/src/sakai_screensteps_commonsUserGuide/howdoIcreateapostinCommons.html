How do I create a post in Commons?

Go to the Commons tool.

Select the Commons tool from the Tool Menu of your site.

Add text.

Enter text into the editor.

Add a link. (Optional)

  1. Click the link button (Insert a link).
  2. Enter a caption for the link in the Display text field.
  3. Enter the URL in the Link to field.
  4. Optionally activate Load thumbnail to generate a small thumbnail preview of the linked page.
  5. Click Insert.

Add an image. (Optional)

  1. Click the image button (Insert an image).
  2. Click Choose File to locate and attach an image file from your device.
  3. Click Insert.

Click Post.
