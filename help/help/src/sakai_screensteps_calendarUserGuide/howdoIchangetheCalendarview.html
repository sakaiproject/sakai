How do I change the Calendar view?

In Calendar, you can view your calendar by day, week, month, year, or even as a list of events. You can also set the default view for your calendar.

Go to Calendar.

Select the Calendar tool from the Tool Menu in your site.

Select the desired view.

Calendar by Day

The item details will display.

Calendar by Week

Calendar by Month

List of Events

Note: When viewing the calendar in list format, you also have the option to select a custom date range and filter events by start/end dates.

Set default view of calendar.

If you have a preferred view (e.g., weekly view), click the Set as Default View button while viewing the calendar in your preferred view.
