What is Become User?

The Become User tool is an administrative tool to allow a user to log in as another user without needing a password. Logging in as another user is useful for support situations, since you’ll be able to see the system from the point of view of that user.

When you log out after using the Become User tool to impersonate/masquerade as a user, you will automatically revert back to your admin account.

To access this tool, select Become User from the Tool Menu in the Administration Workspace.
