How do I add a rubric to an assessment (i.e., test or quiz)?

Connect a rubric with individual instructor-graded questions on an assessment from Tests & Quizzes. Rubrics can be attached to the following question types:

  • File Upload
  • Short Answer/Essay
  • Student Audio Response

Go to Tests & Quizzes.

Select the Tests & Quizzes tool from the Tool Menu of your site.

Create a new assessment or edit an existing assessment.

  1. Click Add to create a new assessment. See How do I create an assessment in Tests and Quizzes? for more information.
  2. Click Actions > Edit to add a rubric to an existing assessment. See How do I edit a published assessment? for more information.

Add or Edit an instructor-graded question type.

  1. While creating or editing an instructor-graded question type, set the Answer Point Value to the same maximum value allowed by the rubric.
  2. Select Use the following rubric to grade this assessment and then choose the rubric.

Configure the additional rubric options. (Optional)

  1. Activate Adjust individual student scores if you would like to be able to change the number of points awarded for individual criteria ratings while grading student submissions on a per-student basis.
  2. Activate Hide Rubric from student if you do not want students to see the rubric prior to submitting.

Click Save.
