How do I grade discussions?

Grade discussions in context within the Discussions tool. Note that you must first create the Gradebook item, and it is helpful to associate the Topic with the Gradebook item to speed up the grading steps.

Go to Discussions.

Select Discussions from the Tool Menu of your site.

Click on a Topic.

Click on a conversation.

Click Grade.

Click the Grade option associated with a post that you wish to grade.

Add points and feedback.

  1. Select an item from the Gradebook items menu. Note that the item must already have been created within the Gradebook to appear in the list.
  2. Complete the Grade field by adding points. Ensure that the points align the possible points within the Gradebook item that you've chosen.
  3. Optionally complete the Comments field.
  4. Click Submit Grade.
