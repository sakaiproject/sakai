How do I post to a topic?

Go to Discussions.

Select Discussions from the Tool Menu of your site.

Click on a Topic.

Click the title of a Topic.

Click Start a New Conversation.

Enter a Title.

Enter a title in the Title field.

Enter a message.

Enter content in the Message field using the Rich Text Editor.

Add Attachments. (Optional)

Optionally attach file attachments to the message. Click the Add attachments button to browse for and select a file from your device.

Click Post.
