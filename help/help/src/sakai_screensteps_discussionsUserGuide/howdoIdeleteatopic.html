How do I delete a topic?

Go to Discussions.

Select the Discussions tool from the Tool Menu of your site.Â 

Select Delete Topic from the More drop-down menu.

Click the More link next to the topic you want to delete, then chose the Delete Topic option from the drop-down menu.

Confirm deletion.

You will be prompted to confirm the deletion. If you are sure that you want to delete the topic and all of the messages within it, click the Delete Topic button.
