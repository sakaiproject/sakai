How do I organize forums and topics?

Forums may be rearranged to change the order in which discussions are listed.

Go to Forums.

Select the Forums tool from the Tool Menu of your site.

Select the Organize button

Select the appropriate number next to the Forum or Topic

Zoom

The numeric pull down menu allows for reordering the Forum, Topic or both. Select the appropriate number for the order the content should appear.  

Click Save

After you placed the forums and topics in the desired order, click Save.
