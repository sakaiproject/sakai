How do I restore a deleted assessment?

Note: Deleted assessments are not copied from one site to another when you import your content into another site.

Go to Tests & Quizzes.

Select the Tests & Quizzes tool from the Tool Menu of your site.

Select the Trash tab.

Select the check box for the assessment(s) you would like to restore.

A list of all your deleted assessments in this site will appear.

Tip: If you have a lot of deleted items, you may use the Search function to locate the one you want to restore.

Select Restore

The restored assessment will appear in your assessment list again.
