How do I export a question pool?

A question pool is a set of questions of any type that belongs to the author and others with whom the author shares it. See What is a Question Pool?

A question pool is available to the author in any worksite in which that person has permission to create assessments, and will show in the list of Question Pools.

You can export question pools from your site to use in another site or system.   

Go to Tests & Quizzes.

Select the Tests & Quizzes tool from the Tool Menu in your site.

Click Question Pools.

Select Export.

Select the Export link below the title of the pool you would like to export.

Choose your export format.

The format you choose will be dependent upon where you intend to import the assessment for use elsewhere.

Click Export.

The question pool exported and downloaded to your local computer.

The exact steps to download the export file may vary depending on the format you select and your browser settings.
