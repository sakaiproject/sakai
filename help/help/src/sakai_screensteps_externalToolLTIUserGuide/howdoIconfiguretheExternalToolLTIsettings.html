How do I configure the External Tool (LTI) settings?

External Tools are often not configured automatically. Use Edit to do so. Site members who are not site owners cannot access the configuration screen for the tool.

Go to External Tool (LTI).

Select External Tool from the Tool Menu of your site.

Click Edit.

Configuring the External Tool settings

The site owner can set up a link to any external tool by configuring the External Tool (LTI) manually. When you make arrangements with an external tool provider, they will provide you with a launch URL for the tool, key, and a secret. Enter these values as Remote Tool Url, Remote Tool Key, and Remote Tool Secret respectively. Once you have entered these values into the tool, it can properly sign launch requests.

Caution: Do not share these values with other members of the site.

Configure the Display Information.

  1. Complete the Set Button Text and Set Tool Title fields.
  2. Optionally, click Choose an icon for this tool to select an custom icon for the external tool.

Configure Gradebook settings.

To configure the external tool to send information to the Gradebook, configure the items in Routing Grades to the Gradebook.

  1. Enter a custom item in Create Gradebook Item.
  2. Or, open Select Gradebook Item to select an existing item from your site.

Configure Launch Window Information.

  1. Check Open in a New Window to open the tool in a new window.
  2. Enter an iFrame Height in pixels to set the frame to a particular size.
  3. Activate Debug Launch to displays the launch data each time the external tool is used. This feature is helpful if troubleshooting the tool configuration.

Release user information.

The external tool configuration also includes the ability to send or suppress user identity information to the external tool, such as the user's name or email address. Check Send Names to the External Tool, Send Email Addresses to the External Tool, and/or Allow the External Tool to retrieve the course roster to release this information to the external tool.

Storing Tool Settings (Optional)

This option allows the external tool to store and retrieve its own settings.

Splash Screen (Optional)

Add information to the Splash Screen field to display custom text to users before they are redirected to the external tool.

Note: This field is plain text only. HTML is not allowed.

Custom Launch Parameters (Optional)

Set custom launch parameters in the Custom Launch Parameters field.

Click Update Options to save your configuration settings.
