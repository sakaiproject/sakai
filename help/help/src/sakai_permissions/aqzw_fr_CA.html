Contrôler l'accès à votre site de cours ou de projet à partir de la Configuration de Sites

Note: Pour compléter les instructions contenues dans ce document, vous devez avoir un rôle ayant les droits d'accès requis. Pour déterminer votre rôle, suivez les instructions dans Rôles du participant. Pour une compréhension générale des rôles et droits d'accès, consultez Droits d'accès et rôles: Vue d'ensemble.

À l'aide de la Configuration de Sites, vous pouvez contrôler l'accès à votre site de cours ou à votre site de projet en suivant les étapes suivantes:

  1. Dans le menu de votre Zone de Travail, cliquez Configuration de Sites, puis cochez la case à côté du site que vous désirez réviser.

    Note: Même si vous pouvez cocher plusieurs cases, vous ne pouvez réviser qu'un site à la fois.

  2. Cliquez Réviser, puis Gérer accès.

  3. Sous "Statut du site", choisissez de publier ou non votre site (c.-à-d. le rendre disponible aux participants du site).

  4. Pour les sites de projets seulement, sous "Accès global", choisissez de garder votre site privé, ou de l'afficher dans le répertoire et de permettre le partage de fichiers que vous sélectionnez.

  5. Vous pouvez rendre votre site largement accessible en cochant la case Possible de s'y inscrire pour quiconque a l'autorisation d'ouvrir une session, et en sélectionnant un rôle pour les personnes qui s'abonnent à votre site. Pour plus d'information, consultez Rôles du participant.


  6. Cliquez Mettre à jour.
