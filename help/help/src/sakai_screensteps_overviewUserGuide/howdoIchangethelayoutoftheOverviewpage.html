How do I change the layout of the Overview page?

Instructors may modify the items and layout of the Overview page in course and project sites.

All users may modify the items and layout of the Overview page in their Home sites.

To access this tool, select Overview from the Tool Menu.

Select the Manage Overview link in the top navigation.

Choose Single or Double column layout.

You may select to view the Overview widgets in a single column or a double column layout.

Use the arrows to move widgets to the desired location.

You may move items left, right, up, or down using the arrows.

To remove a widget from the page, select the X next to the widget you want to remove.

Select + to add Available Widgets not currently on the page.

Select Save once you have finished modifying the page to save your changes.
