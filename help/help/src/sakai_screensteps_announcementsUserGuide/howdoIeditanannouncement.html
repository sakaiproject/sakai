How do I edit an announcement?

Go to the Announcements tool.

Select the Announcements tool from the Tool Menu of your site.

Click Edit below the title of the announcement.

Make your edits.

See "How do I add an announcement?" for information about editing an announcement.

View revision history. (Optional)

If you would like to see the history of revisions for this item, select the See revision history link as shown above.

Click Save Changes when edits are complete.
