How do I unpublish announcements in bulk?

Go to the Announcements tool.

Select the Announcements tool from the Tool Menu of your site.

Select the announcement/s you wish to unpublish.

Select the check box in the Select column for each announcement you would like to unpublish (i.e. return to Draft state).

Select Unpublish.

Select the Unpublish button to unpublish the selected announcements.

Confirm unpublish.

Click Unpublish again to confirm the return to draft state.
