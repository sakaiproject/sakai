How do I view announcements?

You can view your announcements from all sites in your Home site via the Overview, Dashboard, or Announcements tools. The Notifications tool in the global navigation also displays announcements from all of your sites.

You may view announcements for an individual course or project site via the Overview, Dashboard, and/or Announcements tool within that site.

Viewing announcements in Home using Dashboard

The Dashboard tool in your Home site displays announcements from all your sites.

Go to the Dashboard tool.

While you are logged in to Home, select Dashboard from the Tool Menu.

Click on the announcement's View option.

Note: The Dashboard tool in Home will only display announcements from Pinned sites by default.

View announcement details.

Viewing Announcements in Notifications

The Notifications tool displays announcements from all your sites.

Click the notification bell icon in the top global navigation.

If you have new Notifications, you will see a red dot next to the bell icon.

Select Announcements to view all new announcement notifications.

Click on the arrow icon for an announcement to view the item details.

View announcement details.

Viewing announcements in Home using Overview

The Overview area in your Home site displays announcements from all your sites.

Select the Overview tool in your Home site.

The Recent Announcements widget will display announcements from all your active courses.

Click on the announcement subject.

View announcement details.

Viewing announcements in a Course or Project site using Overview

The Overview tool in a Course or Project site displays announcements from that specific site only.

Select the Overview tool in a specific course site.

The Recent Announcements widget will display announcements from the selected course only.

Click on the announcement subject.

View announcement details.

Viewing announcements in a Course or Project site using Announcements

The Announcements tool in a Course or Project site displays announcements from that specific site. Note that the Announcements tool in your Home site will display announcements from all your sites rather than those of a single site.

Note: By default, the instructor view of the Announcements tool within a course displays the editing interface. When you view the Announcements tool as a student (or in your Home site), it displays the read-only view of Announcements.

Go to the Announcements tool.

When you are viewing a course or project site, select the Announcements tool.

Click on the announcement subject.

View announcement details.
