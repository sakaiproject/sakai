How do I search for a site using the Sites tool?

Go to the Sites tool.

Select the Sites tool from the Tool Menu of the Administration Workspace.

Enter your search criteria.

You may search for sites by keyword, site id, or user id (for workspace sites). Once you have entered your criteria, click on the appropriate Search, Site ID, or User ID button to view matching results.

View search results.

All sites matching your criteria will be displayed in a list.
