How do I send a message to a connection in Profile?

Go to Profile.

Select the Profile tool from the Tool Menu in My Workspace.

Click Messages.

Click Compose message.

Enter your message and send.

  1. Start typing the name of one of your connections to get a list of connections you can send a message to, then select them from the list.
  2. Enter a subject.
  3. Enter a message.
  4. Click Send Message.
