How do I locate maxed out caches?

A maxed-out cache will have a count value of 10000 or 100000. Look for these values in the Status area of the Memory tool.

Go to the Memory tool

Select the Memory tool from the Tool Menu of the Administration Workspace.

Click Status.

Look for a count value of 10000 or 100000.

Tip: You can click CTRL+F on your keyboard to use the find on page command in your browser.
