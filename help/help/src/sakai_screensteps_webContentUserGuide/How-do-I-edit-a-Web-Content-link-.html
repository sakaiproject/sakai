How do I edit a Web Content link?

Go to the Web Content link.

Select the Web Content link (e.g. Poetry Magazine) from the Tool Menu in your site. This displays the re-launch page.

Click Edit.

Zoom

Click the Edit icon located in the top right corner of the page.

Make your changes.

Edit any or all of the following:

  • Tool Title
  • Page Title
  • Custom icon
  • Frame Height
  • URL
  • Open in new window

Then, click Update Options to save your changes.

View updated link.

Zoom

The Web Content tool will now display the updated settings.
