What is the Contact Us tool?

The Contact Us tool allows users to quickly and easily reach the appropriate contact for system issues. The user has the opportunity to choose if the issue is content-related, if they need help with the system, if they are having a technical problem, or if they wish to make a feature request. Links specific to each of those areas are automatically configured for each site based on system settings.

Note: This tool, like the Help tool, appears in all Sakai sites by default.

To access this tool, select Contact Us from the Tool Menu in your site.
