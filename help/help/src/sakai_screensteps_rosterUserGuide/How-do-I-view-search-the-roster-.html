How do I view/search the roster?

Note: Depending on your role and permissions, and the individual privacy settings on user profiles, some of the features below may not be available.

Go to Roster.

Select the Roster tool from the Tool Menu of your site.

View list of class participants.

Zoom

Enrolled users in the site will be listed here. You may view the following information about each user:

  1. Name
  2. User ID
  3. Role in the site
  4. Total visits to the site
  5. Last Visit to the site
  6. Connection status (i.e. request a Profile connection)

Searching the roster.

To search the roster for a particular person, type their name or id in the search text box and then click Find.

Filter by group.

Zoom

If you have groups in your site, you may filter by group using the Groups drop-down menu at the top, or the Pick a group drop-down menu to the right of each user's information.

Filter by role.

You may also use the Roles drop-down menu to filter the view by user role.
