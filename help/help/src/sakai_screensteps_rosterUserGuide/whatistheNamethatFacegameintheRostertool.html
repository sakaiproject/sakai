What is the "Name that Face" game in the Roster tool?

The Name That Face game in the Roster tool is a game designed to help faculty learn the names of the students in the class.

The game will present you with photos of the participants enrolled in the site, and you attempt to match each face with a name from the drop-down menu of user's names. If you select the correct name, you can count that item is as "learned" and remove it from this round of the game. If you select an incorrect name, that user is returned to the pool of "unlearned" participants. Unlearned participants will repeat randomly until you have learned all of the users, or until you reset the game.

Note: While this game is primarily intended for faculty, it is also available to students in the course. However, the instructor must first update the Roster permissions to allow students to "View all participants." Otherwise, students will not see the photos in the game and will receive a message that says "There are no students with images on this site, no names to learn."

Go to Roster.

Select the Roster tool from the Tool Menu of your site.

Click the Name That Face button.

Select the "Enter student name..." field to choose a name for the student shown.

Select the Check button to see if you chose the correct name.

Correct response message.

If you selected the correct name, you will see a success message. Click on the I know this student, don't show them to me in this round link to count that student as "learned." Subsequent attempts in this round will only display photos of unlearned users.

Select Continue to proceed to the next student.

Incorrect response message.

If you selected the wrong name, you will receive a message letting you know that it was incorrect. This student will be displayed again randomly until all of the names have been learned, or until you reset the game.

Select Continue to proceed to the next student.

Reroll student.

If you want to go to a different student without selecting a name, you may click on the Reroll student button to try a different student photo.

Reset the game.

Once you have learned all of the names, you will see a Congratulations message letting you know that you've succeeded. You can Reset the game from this screen to start over again if you choose.

You may also choose the Reset the game button at any time during play to reset the students and start counting learned names from scratch.
