How do I view/search the roster?

Note: Depending on your role and permissions, and the individual privacy settings on user profiles, some of the features below may not be available for all users.

Go to Roster.

Select the Roster tool from the Tool Menu of your site.

View list of class participants.

Enrolled users in the site will be listed here. You may view the following information about each user in the site:

  • Picture
  • Name
  • Nickname
  • Pronouns
  • Name Pronunciation
  • User ID (Typically only visible to instructors)
  • Role
  • Total Visits (Typically only visible to instructors)
  • Last Visit (Typically only visible to instructors)
  • Groups (Typically only visible to instructors)
  • Email (Typically only visible to instructors)
  • User Properties (Typically only visible to instructors)
  • Student Number (Typically only visible to instructors)
  • Additional Information (such as Special Needs) Â (Typically only visible to instructors)

Photo Source

Select the desired photo source to view either the official user photo, or the user-uploaded profile image. (If there is no photo for a user, the user's initials will display.)

Roster Views

There are three different roster views available: Cards, Photo Grid, and List. Select the view you would like to see from the Layout area.

Cards

Photo Grid

List

Filter by Group

If you have groups in your site, you may filter by group using the Groups drop-down menu at the top.

Filter by Role

You may also use the Roles drop-down menu to filter the view by user role.

Searching the roster.

To search the roster for a particular person, type his or her name or ID in the search text box and then click Find.
