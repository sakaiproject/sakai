How do I configure MOTD announcement options?

Go to Announcements.

Select the Announcements tool from the Tool Menu in the Administration Workspace.

Click Options.

Zoom

Display options.

These display options control the table/list view settings for the admin user when editing announcements.

RSS Feed Options.

If you would like to subscribe to an RSS feed containing all the public announcements for this site, you may use the URL listed here to subscribe. You may also specify an RSS Alias to make the URL easier to remember.

Display Limits.

The display limits entered here control the default display limits for all users in the system. Individual users may override the default settings if they choose.

Click Update.

Once you have completed your option settings, click the Update button to save.
