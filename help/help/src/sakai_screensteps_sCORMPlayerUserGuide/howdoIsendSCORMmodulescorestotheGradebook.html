How do I send SCORM module scores to the Gradebook?

Select SCORM Player from the Tool Menu in the site.

Note: You must have the Gradebook and SCORM Player tools activated in the site.

Click Configure under the name of the module you wish to connect to the Gradebook tool.

Check the box for Send Score to Gradebook.

Click the Update button to save the settings for this SCORM Module.

This will create an entry in the Gradebook that will automatically update the grades from the SCORM Player.
