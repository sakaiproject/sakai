What is the SCORM Player?

SCORM Player is a tool that allows instructors to incorporate interactive modules onto their course site. Modules built from authoring software, such as Articulate Storyline, Adobe Captivate, or authoring tools are uploaded through the SCORM Player tool and is then accessible to everyone in the site.

Note: SCORM Player currently only supports SCORM 2004, 3rd edition.

To access this tool, select the SCORM Player from the Tool Menu in the site.
