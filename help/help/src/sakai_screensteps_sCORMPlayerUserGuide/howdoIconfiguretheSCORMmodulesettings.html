How do I configure the SCORM module settings?

To access this tool, select the SCORM Player from the Tool Menu in the site.

Click the Configure link for the module you want to edit.

To change the name of the module, enter the new name in the "Name" box.

Set Availability Dates.

  1. To change the date the module is available to students, click on the calendar icon next to "Open Date" and select a date. Then enter the time the module is open in the subsequent boxes.
  2. To change the due date of the module, click on the calendar icon next to "Due Date" and select a date. Then enter the time the module is due in the subsequent boxes.
  3. To change the accept until date of the module, click on the calendar icon next to "Accept Until" and select a date. Then enter the time the module is due in the subsequent boxes.

To change the number of times the students are able to access the module, select the number from the "Number of Tries" dropdown menu.

Set Display Options. (Optional)

Check the box next to the display options you would like to enable:  Show Content Package Structure in SCORM Player and/or Show SCORM Player Controls (e.g. Start, Next, Back, Suspend, Quit buttons)

Enable Grading. (Optional)

If you would like your SCORM module to be scored and the score sent to the Sakai Gradebook, select the check box for Send Score to Gradebook.

Click the Update button once you have made the appropriate changes to your module.
