How do I upload SCORM modules?

To access this tool, select the SCORM Player from the Tool Menu in the site.

Click Upload New Content Package.

Click Choose File and navigate on your computer to find the SCORM-compliant .zip file.

Note: The SCORM Player only supports SCORM 2004, 3rd edition.

Click the Upload File button once you have selected the .zip file and it will be uploaded to Sakai.

View the SCORM package listed in the tool.

Your SCORM content will display in the list.
