How do I download multiple files from Dropbox?

If students have submitted assignments via Drop Box, or if there are a great many uploaded files, use Download ZIP to download all files as one attachment.

Go to Drop Box.

Select Drop Box from the Tool Menu of your site.

Select folders to download.

  1. Select specific folders that you wish to download.
  2. Or, select Check All to select all folders in the site.
  3. Click Download Zip.

Confirm the download.

Click Download to confirm. The ZIP archive will download to your device.
