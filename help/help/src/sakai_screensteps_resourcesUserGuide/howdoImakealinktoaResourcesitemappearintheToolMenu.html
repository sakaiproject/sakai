How do I make a link to a Resources item appear in the Tool Menu?

Instructors can add a link to an important file or folder in the Tool Menu, allowing the link to appear alongside other tools.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Make Web Content Link.

  1. Click the Actions option associated with a file or folder.
  2. Select Make Web Content Link.

Enter a title, and then select Add.

Enter a title for the link in the Title field, and then click Add.
