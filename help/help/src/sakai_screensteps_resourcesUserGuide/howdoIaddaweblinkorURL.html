How do I add a web link or URL?

Create links to web pages in the Resources tool. Links will appear alongside other files and media

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Add Web Links (URLs).

  1. Click the Actions option associated with a folder.
  2. Select Add Web Links (URLs).

Enter the web link information.

  • In the Website Name field, enter the title of the link that will display for site participants.
  • In the Web Address (URL) field, enter (or paste) the web site address.
  • Optionally click Add Another Web Link to add additional links.

Click Add Web Links Now.
