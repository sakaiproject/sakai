How do I create a text document?

Create a basic text document for storage and sharing from the Resources tool.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Create Text Document.

  1. Click the Actions option associated with a folder.
  2. Select Create Text Document.

Enter text, then click Continue.

Enter (or paste) the text into the text box, then click Continue.

This displays the details page for the text document.

Enter document information.

Enter a display name for the text document in the File Name field, and optionally enter information about the text document in the Description field. Click Finish to complete creation of the text document.
