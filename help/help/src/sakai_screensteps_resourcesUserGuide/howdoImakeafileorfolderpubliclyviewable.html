How do I make a file or folder publicly viewable?

Site owners can make a file or folder publicly viewable. This means that the file or folder can be viewed by anyone with the file or folder's URL, even if they're not members of the folder's original site. Some institutions also have a "Search Public Course and Project Sites" button on their gateway page that will allow publicly available files and folders to be searched and viewed.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Edit Details.

  1. Click the Actions menu associated with a file or folder.
  2. Select Edit Details.

Make the item publicly viewable, and then click Update.

  1. Move down the page to locate Availability and Access. Click Availability and Access to expand the options.
  2. Select This file is publicly viewable.
  3. Click Update.

The item is designated becomes publicly available.

Note: While Public content can be viewed by those who are not members of the folder's originating site, the content cannot be searched for or viewed by those who are not logged into your institution's system.
