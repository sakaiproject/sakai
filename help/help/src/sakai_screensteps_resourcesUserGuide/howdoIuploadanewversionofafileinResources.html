How do I upload a new version of a file in Resources?

If a file is linked and active elsewhere in your course site - such as in a Lessons module or an image in a quiz - deleting it from Resources and uploading a new file in its place will break the link. Instead, upload a new version to keep existing links intact.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Upload New Version.

  1. Locate the file that you wish to replace and then click its associated Actions option.
  2. Select Upload New Version.

Click Choose File.

Click Choose File to locate and select the file on your computer.

Click Upload New Version Now.

Ensure that the file name of the new file is displayed in the Choose File field, then click Upload New Version Now.

Note the new version of the file.

Note: The display name for the new file remains the same as the original file.
