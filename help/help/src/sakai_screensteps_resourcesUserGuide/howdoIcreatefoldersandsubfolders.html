How do I create folders and subfolders?

Create new top-level folders which appear to site participants when they first visit the Resources tool, or optionally create subfolders which organize inside of other folders.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Creating folders

Click Actions, and then select Create Folders.

  1. Click Actions, which is to the right of the site's root folder.
  2. Select Create Folders.

Enter the name of the folder.

Enter a title for the folder in the Folder Name field.

Add multiple folders. (Optional)

Click Add another folder to optionally create more than one new folder.

Enter titles for the folders in their associated Folder Name fields.

Click Create Folders Now.

To create the folder(s) in Resources, click Create Folders Now.

Creating subfolders (Optional)

For additional organization, create subfolders inside of folders.

Click Actions, and then select Create Folders.

  1. Click the Actions option associated with a folder.
  2. Select Create Folders.

Enter a title for the subfolder.

Enter a title for the subfolder in the Folder Name field.

Create multiple subfolders. (Optional)

Click Add another folder to optionally create more than one new subfolder.

Enter titles for the subfolders in their associated Folder Name fields.

Click Create Folders Now.

To create the folder(s) in Resources, click Create Folders Now.
