How do I allow all students to upload content to a specific folder?

Give all site participants the technical permission to upload and edit files in a specific folder.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Edit Folder Permissions.

  1. Click the Actions option associated with a folder.
  2. Select Edit Folder Permissions.

Modify student permissions, and then click Save.

In the student column, select Create resources, Edit own resources, and Delete own resources, and then click Save.

These steps can also be used to enhance the permissions that Teaching Assistants possess within the particular folder.
