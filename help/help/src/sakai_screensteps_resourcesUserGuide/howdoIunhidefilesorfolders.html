How do I unhide files or folders?

If you have hidden a file or folder from site participants but wish to make it available once again, use one of the methods in this article to do so.

  • Method 1: Select files or folders > Show
  • Method 2: Actions > Edit Details > Show

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Method 1: Select the hidden file(s) or folder(s), and then click Show.

  1. Check the box associated with a folder or file that you wish to unhide.
  2. Click Show.

Click Show once again to confirm.

Method 2: Click Actions, and then select Edit Details.

  1. Open the Actions menu associated with a file or folder that you wish to unhide.
  2. Select Edit Details.

Select Show this item, then click Update.

  1. Move down the page until you locate Availability and Access. Click Availability and Access to expand the options.
  2. Activate Hide this folder and its contents.
  3. Click Update.
