How do I reorder files or folders within Resources?

Setting an order in which files and folders appear will affect the experience of all site participants. If an instructor wishes students to go directly to Resources to locate content items, this feature allows the items to be placed in a specific order.

The process is the same for reordering both files and folders.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Reordering folders

Click Actions, and then select Reorder.

  1. Click the Actions option associated with the root folder.
  2. Select Reorder.

Reorder items and Save.

Drag and drop with your mouse or use keyboard U and D keys to reorder a folder, then click Save.

View the reordered folder.

Note: If you have used Make a Web Content Link to place a folder in the Tool Menu, the reorder feature explored on this page controls the order of items in the linked folder.

Reordering files in a folder

Click Actions, and then select Reorder.

  1. Click the Actions option associated with a folder containing files. This can include the root folder.
  2. Select Reorder.

Reorder items and Save.

Drag and drop with your mouse or use keyboard U and D keys to reorder a file, then click Save.

View the reordered file.
