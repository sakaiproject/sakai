How do I copy a file or folder from one site to another site?

Instructors can copy a Resource file or folder from on site to another site.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Copy Content from My Other Sites.

Click Copy Content from My Other Sites while viewing the site to which you wish to import files or folders.

Select the site containing the files that you would like to copy.

Click the folder icon (Open this folder) associated with the site from which you would like to copy files or folders.

Select the files or folders you would like to copy and then click Copy.

While navigating content of the site from which you would like to copy content:

  1. Check the boxes associated with files or folders you would like to copy.
  2. Click Copy.

Click Paste.

While once again viewing the files and folders in the site to which you wish to copy, click the Paste option associated with a folder.

View copied item(s) in the new location.

Copied items become available in the new location.
