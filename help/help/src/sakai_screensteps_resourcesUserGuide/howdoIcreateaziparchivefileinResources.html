How do I create a zip archive file in Resources?

Compress a series of files into a single compressed file.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Compress to Zip Archive.

  1. Click the Actions option associated with a folder containing files that you wish to compress.
  2. Select Compress to ZIP Archive.

Note the new zip archive file.

  • The new zip archive file contains a copy of all of the subfolders and files inside the selected folder.
  • The new zip archive file automatically takes the name of the folder that was compressed.
  • By default, the zip file is placed inside of the root folder of the site.
