How do I upload and unpack a zip file to a Resources folder?

Upload a zip file to the Resources tool and then expand within the tool to automatically add all contents.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Uploading the zip file

From the Actions menu, select Upload Files.

  1. Click the Actions option associated with a folder.
  2. Select Upload Files.

Drag and drop the zip file from your computer.

Drag the zip file from your computer and drop it in the Drop files to upload, or click here to browse box.

Or browse your computer for the zip file.

Click the Drop files to upload, or click here to browse button. This action will open your computer's file upload window where you can browse for and select the zip file that you want to upload.

Note the file awaiting upload.

A thumbnail image of the zip file awaiting upload confirmation will display in the Drop files to upload, or click here to browse box.

Click Continue.

Click the Continue button that's near the bottom of the page to confirm the upload of the zip file.

Expanding the zip file

Click Actions, and then select Expand ZIP Archive.

  1. Click the Actions option associated with the uploaded zip file.
  2. Select Expand ZIP Archive.

Note the expanded zip contents.

The zip file is automatically unpacked within the current folder.

  1. A new subfolder within the current folder is created using the name of the zip file.  
  2. The zip file content is unpacked within this new subfolder.
  3. The original zip file remains.
