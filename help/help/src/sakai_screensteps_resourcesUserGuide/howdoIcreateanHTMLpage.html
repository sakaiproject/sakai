How do I create an HTML page?

Create an HTML page for storage and sharing from the Resources tool.

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Click Actions, and then select Create HTML Page.

  1. Click the Actions option associated with a folder.
  2. Select Create HTML Page.

Use the Rich Text Editor to add text, links, and media.

Use the Rich Text Editor to add text, add text, links, and media to the HTML page.

Click Continue to save the HTML page.

Enter document details.

Enter a display name for the HTML page in the File Name field, and optionally enter information about the text document in the Description field. Click Finish to complete creation of the HTML page.

Note the new HTML page in Resources.

The HTML page is available to site participants.
