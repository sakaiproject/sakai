How do I copy a file or folder within Resources in the same site?

Go to Resources.

Select the Resources tool from the Tool Menu of your site.

Select one or more items, and then select Copy.

  1. Check the box associated with the item(s) you want to copy.
  2. Click Copy. If you are copying a single item, you can also use Actions > Copy.

Click Paste.

Click the Paste option associated with the folder to which you wish to copy items.

View the copied item(s) in the new location.

Copied items become available in the new folder.
