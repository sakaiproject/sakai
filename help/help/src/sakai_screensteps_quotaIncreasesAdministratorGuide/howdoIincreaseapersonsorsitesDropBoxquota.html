How do I increase a person's or site's Drop Box quota?

Log in as admin.

Log in as an admin user.

In the Administration Workspace, go to the Sites tool.

Find a course or project site.

Find the site where you would like to increase the Drop Box tool quota.

Select the Go to Site link to access the site.

Select the Go to Site link represented by the icon to the left of the site ID in the search results. The site will display.

Go to the Drop Box tool.

Select Drop Box from the Tool Menu in the site.

Enable the custom quota.

You can enable a custom quota for a specific site participant (e.g., one student in a course site needs their quota increased) or for all users (e.g., an instructor wants every student in the site to have a quota greater than 1 GB per person).

Enable a custom quota for one user only.

Edit Details on the user's folder.

  1. Select the Actions menu for the user's folder.
  2. Select Edit Details.

Enter the new quota.

  1. Select the Special quota checkbox.
  2. In the box provided, enter the new quota in Kilobytes (binary). For quick reference, here is a list of custom Drop Box quotas in Kilobytes:
    • 2 GB = 2097152 KB
    • 4 GB = 4294967 KB

Select Update.

Enable a custom quota for all users in the site.

Edit Details on the top level Drop Box folder.

  1. Select the Actions menu for the main, or top level, Drop Box tool folder.  The folder's name should be [site title] Drop Box.
  2. Select Edit Details.

Enter the new quota.

  1. Select the Special quota checkbox.
  2. In the box provided, enter the new quota in Kilobytes (binary). For quick reference, here is a list of custom Drop Box quotas in Kilobytes:
    • 2 GB = 2097152 KB
    • 4 GB = 4294967 KB

Select Update.
