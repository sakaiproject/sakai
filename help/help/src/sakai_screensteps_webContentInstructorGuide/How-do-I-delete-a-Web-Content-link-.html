How do I delete a Web Content link?

Go to Site Info.

Select Site Info from the Tool Menu in your site.

Click Edit Tools.

Zoom

Un-check the Web Content tool link.

Zoom

Remove the check mark next to the Web Content tool link that you want to delete.

Click Continue.

Click Finish.

Or, click Page Order.

Zoom

Click the red X.

Zoom

Click the red X next to the item you want to delete.

Confirm deletion.

If you are sure you want to delete the item, click OK.

Click Save.
