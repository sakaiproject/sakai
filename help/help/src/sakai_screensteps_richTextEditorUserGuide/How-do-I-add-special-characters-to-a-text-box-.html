How do I add special characters to a text box?

Position the cursor.

Zoom

Position your cursor where you want to insert the special character or diacritical mark.

Click the Insert Special Characters icon.

Zoom

This displays the Select Special Character box.

Select the special character or diacritical mark you want to insert.

View special character.

Zoom

The special character / diacritical mark will now be displayed in the text box.
