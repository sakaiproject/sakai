How do I add/edit a table in a text box?

Position the cursor.

Zoom

Position your cursor in the text box where you want the table to display.

Click Table icon.

Zoom

This displays the Table Properties dialogue box.

Set the number of Rows, Columns and any other table properties needed.

Click OK.

View the table.

Zoom

Edit the table properties. (Optional)

Zoom

To edit the table properties, right-click (CTRL-click - MAC) on the table.

This displays the Edit Table dialog box.

Select the Table Element that you want to edit (Cell, Row, Column, Table or Delete).
