How do I embed an mp3 audio in a text box?

In the text box, position your cursor where you want to embed the mp3 audio file.

Click Insert/Edit Movie.

This displays the Movie Properties dialog box.

Click Browse Server.

This displays the Entity Picker dialog box.

Click the Upload file icon.

Click on the Resources folder for the site you are in. Then click on the Upload File icon to upload the mp3 file. If the file is already in Resources, click on the file and then choose OK.

Drop files or paste URLs or clipboard images.

Or click Select files.

This displays an upload file dialog box.

Select the mp3 audio file you want to embed in the text box.

Once you have located and selected the audio file, click Open to upload it.

Click OK.

Once the file finishes uploading, it will show in the file directory listing and will be selected by default. Click OK to continue.

Check the boxes for Responsive width and Show controls, and then click OK.

View the embedded audio file in the editor.

The editing display will show the embedded audio player.
