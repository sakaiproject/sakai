How do I set empty cells to zero for a single grade item?

Set a score for all empty cells for a specific Gradebook item using Set Score for Empty Cells option. This is fast method of grading students that have not completed a course component.

Note: Gradebook does not include ungraded items in the course grade calculation. Empty cells could result in a higher course grade average for a student. To include a Gradebook item in the course grade calculation, you must enter a grade of zero or higher. Â 

Caution: Set Score for Empty Cells cannot be undone. If you've used the feature in error, it is still possible to manually adjust scores.

Click the arrow icon and then select Set Score for Empty Cells.

  1. Click the arrow icon (Open menu).
  2. Select Set Score for Empty Cells.

Enter a score and then click Done.

  1. Enter a score in the Grade Override field.
  2. Click Done.

Click Continue to confirm.

Click Continue to confirm the score and section or groups you have chosen to update.
