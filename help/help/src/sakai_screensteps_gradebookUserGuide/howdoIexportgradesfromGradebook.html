How do I export grades from Gradebook?

Go to Gradebook.

Select Gradebook from the Tool Menu of your site.

Click Import/Export.

Choose an export method.

  1. Click Export Gradebook to download a CSV file with default configurations.
  2. Optionally click Custom Export to customize the Gradebook data you wish to export.

If using Custom Export, choose the items you would like to include.

  1. Select the data that you would like to export.
  2. Optionally filter students by course section or site group.
  3. Click Download Custom Export.
