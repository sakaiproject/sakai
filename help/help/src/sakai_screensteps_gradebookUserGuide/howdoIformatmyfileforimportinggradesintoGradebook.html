How do I format my file for importing grades into Gradebook?

Manage grades offline in a spreadsheet and then import the file to update the Gradebook. You can also import a file as a method for quickly creating several Gradebook items at once.

To obtain a file in the proper import format, export your Gradebook and use that file as a template. This will ensure that the file contains all of the student information in your site, as well as any existing Gradebook item information in your site.

Go to Gradebook.

Select Gradebook from the Tool Menu of your site.

Click Import/Export.

Click Export Gradebook.

Click Export Gradebook to download a CSV file that you can later use for import.

Open the template in your preferred spreadsheet application.

Add columns, grades, and/or feedback as needed, and save.

The import file must be in CSV file format. Also, be sure to use the following conventions:

  1. Student ID and Student Name are the first two columns and must be retained for any future imports.
  2. Gradebook Items/Assignments can include points by wrapping the points in [ ] after the title (e.g., Assignment 1 [50]).
  3. Comments can be imported by prefixing the column with a * (e.g., * Assignment 1).
  4. Columns that cannot be re-imported are prefixed with #.

Note: add new items to the Gradebook by including new columns in your file. If you add new columns without a point value in brackets, Gradebook will prompt you to indicate the number of points upon import.

In Gradebook, click Choose File.

  1. Click Choose File to browse for and select the import file from your device.
  2. Click Continue.
