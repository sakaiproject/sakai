How do I add items to Gradebook?

Manually add gradable items to the Gradebook to automatically link the grades to each student in the course.

Note that Assignments, Tests & Quizzes, and Student Content in Lessons can be sent to the Gradebook from within the respective tools when the item is posted or published. There is no need to add those items manually in the Gradebook.

Instructors typically add items to the Gradebook manually for things such as:

  • Posting grades for items completed outside of Sakai
  • Discussion Forum grades
  • Participation

Manually adding new items

Go to Gradebook.

Select Gradebook from the Tool Menu of your site.

Click Add Gradebook Item.

Configure the item.

  1. Enter the item title in the Title field.
  2. Enter the item's maximum available points in Point value.
  3. Optionally activate Extra credit. See How does extra credit work in Gradebook? for more information.
  4. Optionally attach a rubric to the item. See How do I add a rubric to a Gradebook item? for more information.
  5. Optionally configure Due date.
  1. Open the Category menu to attach the item to a Category. See How are grades calculated in Gradebook? for more information about Categories.
  2. Activate Release item to students? to make the item and the grades that the item contains available to students immediately.
  3. Activate Include item in course grade calculations? to make the item of the final course grade calculation for students. Leaving the box unchecked omits it from the course grade calculation.
  4. Click Create.

Items Not Included in Grade Calculation

Items which are not included in the grade calculation will display a crossed-out calculator icon (Gradebook item not included in course grade calculation).

Items Not Released to Students

Items which are not released (visible) to students will display a crossed-out eye icon (Gradebook item is not visible to students).

Confirm the new item.

After you have added items to your Gradebook, you will be able to view a list of all Gradebook items on the Grades tab.

Viewing items added by tools

Gradebook items that were added by other tools like Assignments or Tests & Quizzes will display a padlock icon (Go to Assignments tool to edit scores for this item). To edit scores in these cells, graders must visit the tool that has sent the score to the Gradebook.
