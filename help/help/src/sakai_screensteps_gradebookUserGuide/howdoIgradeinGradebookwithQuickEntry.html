How do I grade in Gradebook with Quick Entry?

Use the Quick Entry area of the Gradebook to enter grades and comments for a single item in a simplified interface.

Unlike the standard grading area within the Gradebook, scores and comments entered in the Quick Entry area are only saved when the grader uses Update.

Go to Gradebook.

Select Gradebook from the Tool Menu of your site.

Click Quick Entry.

Select a Gradebook item.

Open the Gradebook item menu to select an item that you wish to grade.

Enter feedback.

  1. Enter scores within the Grade field for each student.
  2. Enter comments within the Comment field for each student.

Excuse a student from the assignment. (Optional)

Check the Excused box assocaited with a student to excuse them from the assignment.

See How do I exclude a grade for a specific student? for more information and for alternate methods of excusing students.

Set a score for empty grades. (Optional)

Click Set Score For Empty Grades to set the same grade for all students that do not yet have a grade.

  1. Enter a score in the Set empty scores to field.
  2. Click Set.

Set a comment for empty comments. (Optional)

Click Fill Empty Comments to set the same comments for all students that do not yet have a comment.

  1. Enter comments in the Set empty comments to field.
  2. Click Set.

Click Update.

Move down the page until you locate Update. Click Update to save the feedback.

Caution: feedback will not save automatically! Click Update to save.
