How do I import grades into Gradebook?

Import grades as a batch from a separate file. See How do I format my file for importing grades into Gradebook? for information about preparing the import file.

Go to Gradebook.

Select Gradebook from the Tool Menu of your site.

Click Import/Export.

Click Choose File.

  1. Click Choose FileÂ to browse for and select the import file from your device.
  2. Click Continue.

Note: The import file must be in a specific format in order to import correctly. It should be a .CSV file and contain the appropriate student and Gradebook item data. See How do I format my file for importing grades into Gradebook? for information about preparing the import file.

Select the items to be imported.

A summary of Gradebook items with expected changes will be previewed for you. Check the box next to each of the items you would like to create or update. Click Next to continue with the import.

Configure the new Gradebook item for import. (Optional)

  1. If you are creating new items as part of the import, you will be prompted to edit the item options for each new item. See How do I add items to the Gradebook? for complete information about configuring new items.
  2. Click Next.

Confirm the details and click Finish.
