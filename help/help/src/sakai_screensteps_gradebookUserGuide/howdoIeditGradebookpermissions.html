How do I edit Gradebook permissions?

By default, only instructors and site owners can enter feedback in Gradebook. Use the Permissions area to provide Teaching Assistants with grading permissions for specific groups, course sections, or all students in the site.

A user in the Teaching Assistant role must be present in the site before you can take these steps.

Go to Gradebook.

Select Gradebook from the Tool Menu of your site.

Click Permissions.

Select a grader.

Open Select a grader to edit to select a user in the site with the Teaching Assistant role.

Click Add Rule.

Configuring the new rule

Choose View or Grade.

In the Can menu, select View or Grade.

  • Select Grade if you intend for the Teaching Assistant to enter scores and feedback.
  • Select View if you intend for the Teaching Assistant to have read-only access to scores and feedback.

Choose Categories.

Open the Category menu and select a Category that you'd like the Teaching Assistant to access. Select All Categories if you'd like the Teaching Assistant to have access to all items in the course.

Choose sections and groups.

Open the Group menu and select a group or course section of students that you'd like the Teaching Assistant to access. Select All Sections/Groups if you'd like the Teaching Assistant to have access to all students in the course.

Release the current course grades to the Teaching Assistant. (Optional)

Optionally check Allow grader to see course grade for selected section(s) below to allow the Teaching Assistant to view the current course grade for selected students.

Click Save Changes.

Note: Create additional Rules if you'd like a Teaching Assistant to have different permissions for specific groups/course sections, or if you have more than one Teaching Assistant that to whom you'd like to provide access to the Gradebook.
