How do I edit a podcast?

Go to Podcasts.

Select the Podcasts tool from the Tool Menu of your site.

Click Edit link for podcast item.

Click Edit link in display of podcast item you want to edit.

Edit podcast file or item data and save changes.

  1. If podcast file needs to be updated, click the Choose File button and select the file from the pop-up dialog box.
  2. Edit item Publish Date/Time, Title, or Description.
  3. Click Save Changes button.
