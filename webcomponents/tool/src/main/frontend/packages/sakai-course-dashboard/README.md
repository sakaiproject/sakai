# sakai-course-dashboard

## Installation

```bash
npm i @sakai-ui/sakai-course-dashboard
```

## Usage

```html

<sakai-course-dashboard></sakai-course-dashboard>

```

To scan the project for linting and formatting errors, run

npm run lint
