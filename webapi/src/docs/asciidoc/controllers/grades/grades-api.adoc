== Grades

__See the login api about session cookies__

=== GET /api/users/me/grades
Returns grades for any assignments that the current user can view, across all of their pinned
sites. If the user has grading permissions on any of their pinned sites then the grades for all
users in that site will be returned.

[cols="1h,3", grid=rows, frame=topbot]
|===
| Authentication | User must be authenticated
| Produces       | application/json
|===

==== Example Request
include::{snippets}/get-user-grades/curl-request.adoc[]

==== Responses
[cols="1,2,2", options="header"]
|===
| Status Code | Status | Details
| 200 OK | Success | Returns a JSON document with the sites and viewable grades for the current user
| 401 Unauthorized | Error | User is not authenticated.
|===

include::{snippets}/get-user-grades/response-body.adoc[]


=== GET /api/sites/{siteId}/grades
Returns grades for any assignments that the current user can view, in the supplied site. If the
current user has grading permissions on the site, then the grades for all the users in the supplied
site will be returned.

[cols="1h,3", grid=rows, frame=topbot]
|===
| Authentication | User must be authenticated
| Produces       | application/json
|===

==== Example Request

==== Responses
[cols="1,2,2", options="header"]
|===
| Status Code | Status | Details
| 200 OK | Success | Returns a JSON document with the viewable grades in the supplied site, for the current user
| 401 Unauthorized | Error | User is not authenticated.
|===
