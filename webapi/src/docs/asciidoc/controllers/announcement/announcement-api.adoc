== Announcement

__See the login api about session cookies__

=== GET /api/users/me/announcements
Returns announcements for the currently authenticated user across all their sites.

[cols="1h,3", grid=rows, frame=topbot]
|===
| Authentication | User must be authenticated
| Produces       | application/json
|===

==== Example Request
include::{snippets}/get-user-announcements/curl-request.adoc[]

==== Responses
[cols="1,2,2", options="header"]
|===
| Status Code | Status | Body / Details
| 200 OK | Success | Returns an announcements object containing a list of announcement entries.
| 401 Unauthorized | Error | User is not authenticated.
|===
