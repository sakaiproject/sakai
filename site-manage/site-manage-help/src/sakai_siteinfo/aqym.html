Site Info: Overview

What it does

The Site Info tool provides information about the current site. This tool's primary purpose is to allow site managers to update the site's settings and membership.

For site participants

  • If you have limited permissions, you'll see the site's description and your group memberships, if applicable.

For site managers

  • Site Info allows you to change information about the site, tools available in the site, and access to the site.

  • The functions of the Site Info tool are also available through the Worksite Setup tool, which is available on the menubar when you are in My Workspace.

  • Tip: If the warning "Unpublished site" appears above the site menu, the site is invisible to participants. Once you publish the site, your participants will be able to see it as a tab or find it in the list of active sites.
