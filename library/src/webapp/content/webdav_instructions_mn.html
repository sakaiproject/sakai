  • Windows Vista/Windows 7

Windows Vista/Windows 7-д WebDAV-ын тохиргоо хийх

Анхааруулга:Windows Vista дахь Сакайн WebDAV-г хэрэглэхийн тулд Vista Service Pack 2-ыг суулгасан байх ёстой. Хэрэв та одоохондоо Service Pack 2-ыг суулгаж амжаагүй бол Start > All Programs > Windows Update-ыг дарж Windows Update-ыг ажиллуулна уу.

  1. Desktop-оос юмуу Start цэсээс Computer-ыг дар.
  2. Цонхны дээд хэсэг дэх Map Network Drive-ыг дар.
  3. Drive drop down цэсээс сул байгаа чиглүүлэх үсгийг (e.g., Z)сонго.
  4. Folder-ын хэсэгт дээр харуулсан энэ сайтын URL-ыг бич (эсвэл хуулаад paste хий). Finish-ыг дар.
  5. Хэрэгтэй бол Сакайнхаа хэрэглэгчийн нэр, нууц үгээ оруулаад OK-ыг дар.
  6. Одоо та файлууд болон фолдеруудаа компютер, сайтынхаа нөөц фолдер хоёрын хооронд чирэх боломжтой.

Тохиргоо хийгдэж дуусахаар сургалт юмуу төслийн нөөцүүдийн фолдер нь Windows Explorer-ын бусад drive-ын адил ажиллах Computer-ын доор drive хэлбэрээр гарч ирнэ. Одоо та файлууд болон фолдеруудаа компютер, сайтынхаа нөөц фолдер хоёрын хооронд чирэх боломжтой.

Windows Vista/Windows 7-д WebDAV-ын тохиргоо хийх өөр нэг арга

Хэрэв дээрх үйлдэл холболтын алдаа үүсгэвэл AnyClient гэж нэрлэгддэг гуравдагч талын клиент компютерыг ашиглан файлуудруугаа нэвтрэхийн тулд WebDAV-ын үйл ажиллагааг ашиглаж болно. Татах, суулгах, болон AnyClient-ыг брдүүлэхийн тулд энэ үйлдлийг дага.

  1. AnyClient WebDAV client-ыг http://www.anyclient.com-аас татаж аваарай.
  2. install.exe файлыг ажиллуулж, тохиргооны Визардыг хийж дуусгана.
  3. Хэрэв суулгах программ AnyClient-ыг автоматаар ажиллуулахгүй бол AnyClient-ыг нээхийн тулд Start > All Programs > AnyClient > AnyClient-ыг дар.
  4. Зүүн доод булан дахь Connect-ын дар.
  5. Зүүн доод булан дахь New-ын дарж зүүн тал дахь цэсэнд энэ холболтод утга учиртай нэр өгөөрэй ( History 101 Resources г.м).
  6. Host харилцах цонхон дахь step 1-д, дээр харуулснаар замаа бичнэ(эсвэл хуулаад paste хийнэ).
  7. Сакайн хэрэглэгчийн нэр, нууц үгээ бичнэ.
  8. Connection type дроп бокст WebDAV-ыг сонгоно.
  9. Дахин хэрэглэх энэ холболтоо хадгалахын тулд цонхны доод хэсэг дэх Save-ыг дараад Connect-ыг дар.

Mac (OS 10.4 - 10.6)-д WebDAV-ын тохиргоо хийх

Анхааруулга: OS X-ын тодорхой шинэчлэлээс хамааран Macintosh илрүүлэгч дэх Webdav-ын тусламж ашиглахад хэтэрхий дутуу дулимаг байж магадгүй. Хэрэв танд энэ тал дээр асуудал гарвал Cyberduck-ыг ашиглаарай.

  1. Илрүүлэгч дэх дэлгэцийн дээд тал дахь цэсээс Go > Connect to Server...-ыг сонго
  2. Server Address харилцах цонхноос step 1-д дээр харуулснаар замаа бичнэ(эсвэл хуулаад paste хийнэ).
  3. Дахин хэрэглэхээр энэхүү замаа хадгалахын тулд + товчийг дар.
  4. Connect-ыг дар.
  5. Хэрэглэгчийн нэр, нууц үгээ бичээд Connect-ыг дар.

Сайт дахь нөөцүүдийг харуулах дэлгэцэн дээр одоо та цонх харах болно. Одоо та энэ цонхноос юмуу энэ цонхруу файлуудыг чирээд тавьж болох бөгөөд файлуудыг хогийн савруу чирж устгах боломжтой.

Linux (Gnome)-д WebDAV-ын тохиргоо хийх

Анхаар:Linux-ын зарим хувилбарууд нь Home-ын нөөцөд нэвтрэхэд асуудал гарч болзошгүй боловч бусад бүх сайтын төрлүүдрүү нэвтрэхэд зүгээр ажиллана (сургалт, төсөл, г.м).

  1. Places > Connect to Server...-руу ор
  2. Service type дахь WebDav-ыг сонго .
  3. Server-д бичнэ(эсвэл хуулаад paste хийнэ): hostname
  4. Port-д бичнэ(эсвэл хуулаад paste хийнэ): 80
  5. Folder-д "Үе шат 1"-ын сүүлийн хэсгийн URL: dav/xxxx-xxxx-xxxx-ыг бичнэ(эсвэл хуулаад paste хийнэ)
  6. Хувилбарыг шалгана Add bookmark
  7. Bookmark name-д "Network"-ын доор үүсгэх фолдерынхоо нэрийг оруулаад Next-ыг дар.
  8. Одоо Connect дээр дар.
  9. Эцэст нь Сакайн нууц үгээ оруулах хэрэгтэй болно.

Одоо та Places-ын доор шинэ drive-ыг харах болно. Энэ нь сайт дахь нөөцүүдийг төлөөлнө. Энэ компютер дээрх бусад фолдеруудтай адилхан ажиллана. Одоо та энэ цонхноос юмуу энэ цонхруу файлуудыг чирээд тавьж болох бөгөөд файлуудыг хогийн савруу чирж устгах боломжтой.

Linux (KDE)-д WebDAV-ын тохиргоо хийх

Анхаар: Linux-ын зарим хувилбарууд нь Home-ын нөөцөд нэвтрэхэд асуудал гарч болзошгүй боловч бусад бүх сайтын төрлүүдрүү нэвтрэхэд зүгээр ажиллана (сургалт, төсөл, г.м).

  1. Файл менежерыг нээ (Dolphin).
  2. Зүүн тал дахь Places-д Network-ыг сонго.
  3. Add network folder дээр дар.
  4. web folder (webdav)-ыг шалгаад Next дээр дар.
  5. Name-д "Network"-ын доор үүсгэх фолдерынхоо нэрийг оруулаад Next-ыг дар.
  6. User-д Сакайн хэрэглэгчийн нэрээ оруул.
  7. Server-д hostname-ыг бичнэ(эсвэл хуулаад paste хийнэ):
  8. Folder-д "Үе шат 1"-ын сүүлийн хэсгийн URL: dav/xxxx-xxxx-xxxx-ыг бичнэ(эсвэл хуулаад paste хийнэ)
  9. Хэрэв URL нь HTTPS-ыг ашигладаг бол Encoded хувилбарыг шалга, порт нь автоматаар 80 болж өөрчлөгдөнө.
  10. Create an icon for this remote folder хувилбарыг шалга.
  11. Одоо Save and connect дээр дар.
  12. Эцэст нь Сакайн нууц үгээ оруулах хэрэгтэй болно.

Одоо та (Dolphin) файл менежер дотор Network-ын доор шинэ drive-ыг харах болно. Энэ нь сайт дахь нөөцүүдийг төлөөлнө. Энэ компютер дээрх бусад фолдеруудтай адилхан ажиллана. Одоо та энэ цонхноос юмуу энэ цонхруу файлуудыг чирээд тавьж болох бөгөөд файлуудыг хогийн савруу чирж устгах боломжтой.

Linux (console)-д WebDAV-ын тохиргоо хийх

  1. Шинэ терминалыг нээ.
  2. Коммандаа бич cadaver URL from Step 1
  3. Та магадгүй гэрчилгээг хүлээн зөвшөөрөх хэрэгтэй болно
  4. Сакайнхаа хэрэглэгчийн нэр, нууц үгээ бич
  5. Одоо та коммандын эгнээнээс баримт татаж, оруулж болно.

Бүх боломжит коммандуудыг авах бол help-ыг дар.
