Sakai へようこそ

Sakai デモへようこそ．このデモ設定は，Sakai インスタンスを素早くかつ簡単に動かし始めるための一つの方法として作成されました． 実運用のための設定に関する詳しい情報については，readme ファイルをご参照いただくとともに，Sakai ウェブサイト sakaiproject.org や Sakai デベロッパサイト collab.sakaiproject.org も訪問してみてください．

ここに表示されている情報は，sakai.properties 設定ファイルの "server.info.url" の値を所望の html ファイルを参照するように Sakai 管理者が編集することによって修正することができます．
