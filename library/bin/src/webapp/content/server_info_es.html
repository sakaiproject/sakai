Bienvenido a Sakai

Bienvenido a la demostración de Sakai. La configuración de demostración se ha creado para tener una instancia de Sakai funcionando de manera rápida y fácil. Para más información sobre la instalación de Sakai como solución, por favor revise los ficheros readme, visite la página principal de Sakai sakaiproject.org, y la página del desarrollador de Sakai en collab.sakaiproject.org.

La información por defecto que aparece aquí para un usuario nuevo se puede modificar por el administrador. Para ello debe editar la propiedad "server.info.url" del fichero sakai.properties para que se muestre el fichero html que se desee.
