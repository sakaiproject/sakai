Сакайд тавтай морил

Сакайн демод тавтай морил. Демогийн тохиргоог танд Сакайгийн амархан, хурдан ажилладаг хувилбарыг авах боломжийг олгох зорилгоор үүсгэсэн. Хэрэгтэй бүтээгдэхүүнийхээ зөв гарцад зориулсан Сакайг суулгах талаар илүү мэдээлэл авах бол readme файлуудыг унших болон sakaiproject.org гэсэн Сакайн вэб сайт болон collab.sakaiproject.org гэсэн Сакай хөгжүүлэгчийн сайтруу ороорой.

Энд харагдаж буй мэдээллийг sakai.properties тохиргооны үнэлгээ "server.info.url" файлыг засах Сакайн администратор өөрчилж болно.
