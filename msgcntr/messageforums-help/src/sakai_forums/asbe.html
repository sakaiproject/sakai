Viewing Forums

To see the list of forums for your site, click Forums in your site's menubar.

Viewing messages

Note: The Forums tool does not contain default forums. No forums will appear here until the site creator (or someone with an appropriate role) creates them.

Each forum appears in an enclosed box with its title and short description displayed (if created). If a full description of the forum exists, click Read Full Description to view it.

Beneath the forum title, topics are listed by title, and include a short description (if created). The following details also appear:

  • A count of total messages
  • A count of unread messages
  • If a topic is moderated, the word "Moderated" will be displayed.
  • If a topic has an attachment, its filename will be displayed. Click the filename to open the attachment or save it to your computer.

Clicking the title of any forum will open an isolated view of a particular forum and its topics.

Clicking the title of any topic will display a thread of messages for that particular topic. If a full description of the topic exists, click Read Full Description to view it. Each individual thread appears in a list with a count of total messages and unread messages below. Use the arrows to the left of the messages to expand the threads as needed.

To expand all threads and messages, click Display Entire Message. To mark all of the messages as read, click Mark All As Read. To open a printable version of the messages, click the printer icon at the top.

Clicking the title of the thread will open an isolated view of a particular thread and its individual messages. Use the View drop-down list to change the display of messages.

To open an individual message, click its subject.

Forum options

Depending on how the forum settings are configured, and your role within the site, the following options may be available:

  • New Forum: Opens the Forum Settings screen for creating a new forum. For more information, see Creating a new forum and topic in Forums.

  • Organize: Opens the Organize Forums and Topics screen for changing the order of forums and topics. For more information, see Changing the order of forums and topics.

  • Template Settings: Opens the Default Settings Template screen. For more information, see Revising or deleting a forum or topic.

  • Statistics: Displays individual statistics for course participants. For more information, see Participation statistics.

  • Pending Messages: For moderated forums, displays any messages pending approval. For more information, see Moderating messages in a forum.

  • New Topic: Opens the Topic Settings screen for creating a new topic. For more information, see Creating a new forum and topic in Forums.

  • Forum Settings: Opens the Forum Settings screen for revising an existing forum. For more information, see Revising or deleting a forum or topic.

  • Topic Settings: Opens the Topic Settings screen for revising an existing topic. For more information, see Revising or deleting a forum or topic.
