ACCESSIBILITAT

Cal tenir en compte les necessitats de les persones amb discapacitats quan es creen continguts web.

Hi ha molta informació sobre la qüestió de l'accessibilitat en crear continguts web. Un exemple és WebAIM's, Introduction to Web Accessibility.

L'eina Lliçons disposa de diverses funcions que faciliten la creació de continguts accessibles:

Imatges

Les imatges han d'incloure una etiqueta "ALT" amb una descripció textual de la imatge. Aquelles imatges que s'inclouen per decoració o per afegir espais buits, etc., han d'anar amb etiquetes TAG buides.

Quan afegiu una imatge fent ús de "Afegeix contingut multimèdia", el botó d'edició us permetrà afegir el text corresponent a l'etiqueta ALT.

Si inseriu una imatge fent ús de "Afegeix text", amb l'editor de continguts web, el botó "Imatge" facilita també afegir el text corresponent a l'etiqueta ALT.

Contingut multimèdia

Si feu ús de continguts multimèdia, per recuperar-los cal proporcionar alternatives que siguin apropiades pels lectors de pantalla.

A l'eina Lliçons hi ha diverses maneres de proporcionar continguts alternatius. La més senzilla és afegir una descripció sota el contingut. Després d'afegir-lo amb "Afegeix contingut multimèdia", el botó d'edició permet afegir un camp de descripció sota el contingut.

Si amb la descripció no n'hi ha prou podeu usar "Afegeix una subpàgina" per crear una pàgina amb un contingut alternatiu més extens, o usar "Afegeix un recurs" per a que apunti a un recurs existent.
