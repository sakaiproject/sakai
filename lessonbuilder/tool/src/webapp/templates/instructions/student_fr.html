Pour commencer avec la fonction "Contenu Etudiant"

Remplissez votre page en utilisant les commandes suivantes :

  • Ajouter Texte - Vous permez de créer un texte et de l'illustrer avec un éditeur
  • Ajouter Ressource - Pour télécharger un document ou pointer vers un site Web
  • Ajouter Multimedia - Pour ajouter un objet qui sera visible dans la page, comme une présentation Flash, par exemple, ou une vidéo
En plaçant la souris sur une commande vous ferez apparaître une courte description.

Lorsque vous aurez ajouté un contenu, vous verrez un bouton "Éditer" auprès de chaque item. Il vous permettra d'ajuster la taille d'un objet comme une vidéo, de changer un titre ou un texte de description. "Éditer" permet également d'organiser le parcours des items de façon séquentielle.
