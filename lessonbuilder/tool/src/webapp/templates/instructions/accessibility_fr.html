ACCESSIBILITE

En créant des contenus, vous devez prendre en considération les personnes handicapées.

Il existe beaucoup d'information disponible sur ce sujet. Vous pouvez par exemple consulter WebAIM's, Introduction to Web Accessibility.

Leçon possède plusieurs dispositifs qui vous aideront à créer un contenu accessible :

Images

Les images doivent contenir, dans le tag "ALT ", une description textuelle de l'image. Seules celles qui sont employées pour la décoration, pour ajouter des blancs ou toute autre élément du décor peuvent posséder un tag ALT vide.

Lorsque vous ajouterez une image avec "Ajouter Multimedia", the bouton "Edit" vous proposera de remplir un texte ALT.

Si vous insérez une image, en utilisant l'éditeur web, avec "Add Text", le dialogue "Image" vous demandera également de remplir un champ ALT.

Contenu Multimedia

Lorsqu'on introduit un contenu multimedia, il est recommandé de fournir un chemin alternatif vers cette ressource, de façon qu'il soit possible, pour certains handicapés, de le lire avec un lecteur spécialisé.

Dans Leçon il existe plusieurs moyens de fournir cette voie alternative. Le plus simple est d'ajouter une description en-dessous de la ressource. Après avoir ajouté une telle ressource, avec "Ajouter Multimédia", il vous suffira d'employer le bouton "Éditer" et de remplir le champ de description de l'objet.

Si une description courte ne suffisait pas, n'hésitez pas à employer "Ajouter sous-page" pour créer une page supplémentaire ou à employer "Ajouter Ressource" pour pointer vers une ressource existante.
