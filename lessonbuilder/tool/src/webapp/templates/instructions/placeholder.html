How to add content to this page

You can add content in two ways:
  • The "Add Content +" button at the top will add a new item at the bottom of the page.
  • The "+" icon to the right of each existing item will add a new item above that one.
Both of these will bring up the Add Content dialog. The most commonly used options in that dialog are
  • Add Text - Add text with a rich text editor.
  • Add Content Links - Add links to files, web pages, or images.
  • Embed Content on Page - Place images or video on the page.
  • Add Assignment - Link to an assignment in the Assignments tool.
  • Add Forum or Topic - Link to a forum or topic in the Forums tool.
  • Add Subpage - Add a new page. A link to it will appear on this page.

You can add other types of content as well. For more information, see Getting Started with Lessons
