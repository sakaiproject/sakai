Funcionament bàsic de l'eina Lliçons

Fent ús de les eines del menú de la part superior es poden afegir continguts a la pàgina. Les més usades són:

  • Afegeix text - Permet introduir text mitjançant un editor.
  • Afegeix un recurs - Permet pujar un recurs o afegir un enllaç a un espai web.
  • Afegeix contingut multimèdia - Permet afegir elements que es mostraran a la pàgina com ara una presentació Flash o un vídeo.

Recomanem començar fent clic sobre Afegeix text. Després podreu afegir enllaços a continguts de l'eina de Recursos, a activitats de l'eina Activitats, a proves de l'eina Tests o a temes de l'eina Debat. Feu clic a Afegeix contingut multimèdia per afegir presentacions Flash, vídeos, enllaços a Youtube, etc.

Al passar el ratolí per sobre les opcions del menú es mostra una descripció breu del que permet fer l'element.

Un cop hagueu afegit contingut veureu botons d'edició per poder ajustar la mida dels objectes multimèdia, canviar els títols, afegir text descriptiu, etc. També podreu editar els elements de manera que la informació es publiqui seqüencialment.

Altres elements que podeu incloure en una pàgina (si no apareixen directament al menú els podreu veure fent clic sobre "Altres eines")

  • Afegeix un espai web - Permet pujar un fitxer ZIP amb els continguts. Permet els formats Camtasia, Wimba Create i Articulate, però en general es pot pujar qualsevol fitxer ZIP tenint en compte certes precaucions (trobareu més informació més endavant).
  • Afegeix una subpàgina - Crea una pàgina secundària que també pot contenir tot tipus d'elements i que estarà enllaçada a l'original.
  • Afegeix una activitat de l'eina Activitats - Permet seleccionar una activitat de l'eina "Activitats" de Sakai i incloure-la a la pàgina.
  • Afegeix una prova de l'eina Tests - Permet seleccionar una prova de l'eina "Tests" de Sakai i incloure-la a la pàgina.
  • Afegeix un tema d'un fòrum de debat - Permet seleccionar un tema de l'eina "Debat" de Sakai i incloure'l a la pàgina.
  • Afegeix una eina externa - Quan sigui necessari usar aquesta funció, rebreu la informació apropiada.
  • Afegeix comentaris - Permet que els estudiants afegeixin comentaris.
  • Afegeix continguts dels estudiants - Permet que els estudiants creïn les seves pàgines amb contingut propi. La resta d'estudiants les podran comentar.

El botó "Paràmetres", a la part superior dreta, permet afegir funcionalitats suplementàries que s'aplicaran a tota la pàgina. El botó d'edició que apareix al costat de cada element permet controlar els paràmetres específics d'aquell element.

Més informació

  • Contingut Multimèdia Instruccions detallades i consells tècnics relatius a diferents navegadors.
  • Accessibilitat Web Per crear contingut accessible amb l'eina Lliçons.
  • Seqüenciació Requerir que es completin tasques abans de publicar-ne de noves.

Usos habituals

Lliçons es pot usar de diverses maneres. Les més habituals són les següents :

  • Organitzar els documents per tema o unitat - Pels temes principals, que segurament voldreu que apareguin al marge esquerra, aneu a Informació de l'espai i creeu una instància nova de l'eina Lliçons usant l'opció Edita les eines. Aquest procés us requerirà de posar un títol a la unitat nova. El podreu canviar després usant el botó per editar el títol.
  • Organitzar els continguts nous - Trobareu un exemple a la part dreta de la pàgina. Hem creat pàgines que contenen informació sobre tres subtemes específics, tots ells accessibles des de la pàgina original mitjançant enllaços. Per crear una pàgina com aquesta feu ús de l'opció Afegeix una subpàgina del menú de la part superior.

    Noteu que "Afegeix una subpàgina" us ubica dins d'aquesta pàgina nova per poder afegir-hi continguts. Podeu tornar a la pàgina original fent ús dels enllaços que hi ha a la part superior.

Ús avançat

Per la majoria de casos n'hi haurà prou amb els dos mètodes descrits. Tot i això és possible usar Lliçons per crear estructures molt més complexes. Presentem altres mètodes possibles :

  • Construir una seqüència de pàgines - Algun cop cal que els estudiants segueixin una seqüència concreta de pàgines,

    Per afegir una pàgina següent, useu Afegeix una subpàgina seleccionant l'opció "pàgina següent". Típicament també voldreu escollir "mostra com a botó".

    Marcar l'opció "pàgina següent" crea una pàgina des de la qual no cal retornar a l'actual. (Els estudiants sempre poden accedir a qualsevol pàgina des de l'índex, si no és que heu restringit l'accés a la pàgina.)

    L'opció "mostra com a botó" crea un botó enlloc d'un enllaç.

  • Accedir a pàgines diferents depenent de l'elecció de l'estudiant - Al marge dret en podeu veure un exemple (cas 3). De vegades segons la resposta a una pregunta es vol que els estudiants accedeixin a pàgines diferents. Per fer-ho, introduïu la pregunta fent ús de l'opció Afegeix text , i afegiu a sota dos elements pàgina següent que apuntin a pàgines diferents. Per fer-ho, feu ús Afegeix una subpàgina dues vegades.

    L'elecció de les opcions de "Afegeix una pàgina" depenen de com es vulgui organitzar l'espai. Podeu veure a l'exemple 3 com s'empren "Mostra com a botó" i "Pàgina següent".



Exemple 1. Pàgina de mostra construïda amb Afegeix text,Afegeix una prova, Afegeix una activitat i Afegeix un tema d'un fòrum. La imatge s'ha afegit amb l'editor web que apareix a Afegeix text.

Exemple 2. Exemple d'una unitat amb diverses seccions: un text curt amb Afegeix text i tres sub-pàgines, una per cada secció. Per afegir continguts a les subpàgines només cal fer clic sobre l'enllaç corresponent.



Exemple 3. Exemple de navegació a pàgines diferents depenent d'allò que escullen els estudiants. Les dues respostes s'afegeixen fent ús de Afegeix una subpàgina. Hi ha dues opcions marcades: "Pàgina següent" i "Mostra com un botó". Cal adonar-se que Afegeix una subpàgina permet crear-ne una de nova o bé fer referència a una d'existent.
