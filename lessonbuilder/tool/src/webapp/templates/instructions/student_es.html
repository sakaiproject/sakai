Introducción al contenido del alumno

Pon tu contenido en esta página usando uno de los siguientes comandos:

  • Añadir Texto - escribe texto en un editor
  • Añadir Recurso - sube un documento o añade una URL a un sitio web
  • Añadir Multimedia - añade un elemento que se mostrará en la página, ej. Presentación en Flash o un vídeo
Si sitúas el ratón encima de un comando, obtendrás una breve descripción de lo que hace.

Cuando hayas añadito contenido, encontrarás el botó Editar al lado de cada elemento donde puedes ajustar el tamaño de los objetos multimedia, cambiar el título, añadir un texto descriptivo, etc. Edita elementos para liberar actividades e información secuencialmente.
