ACCESIBILIDAD

Cuando diseñe sus contenidos, tome la precaución de tener en cuenta las necesidades de las personas discapacitadas.

Existe gran cantidad de información en internet sobre la creación de contenidos web accesibles. Por ejemplo: Introducción a la accesibilidad web.

La herramienta "Diseño de unidades" tiene varias funcionalidades para ayudarle a crear contenidos accesibles.

Imágenes

Las imágenes suelen llevar un "texto alternativo", una descripción textual del contenido de la imagen; aunque las imágenes cuya única finalidad sea decorativa deberían tener un texto alternativo vacío.

Tras añadir una imagen mediante el uso de "Incrustar contenido en la página", el botón "Editar" le permitirá definir el texto alternativo de la imagen.

Si inserta una imagen en un bloque de texto usando el editor de texto, el cuadro de diálogo "Propiedades de Imagen" también le permitirá proporcionar un texto alternativo.

Contenido multimedia

Al utilizar contenido multimedia, deberíamos proporcionar modos alternativos de acceder a él, al menos alguno que sea apropiado para el uso de lectores de pantalla.

En la herramienta que nos ocupa existen diversas formas de proporcionar contenido alternativo. La más sencilla es añadir una descripción bajo el contenido. Tras añadir contenido con la opción "Incrustar contenido en la página", el botón "Editar" le permite añadir dicha descripción.

Si una breve descripción no es suficiente, podría usar "Añadir subpágina" para crear una páagina con un contenido alternativo más extenso, o usar "Añadir Enlace a contenido" para enlazar a un recurso existente.
