Lessons: Overview

Introduction

Lessons is a way to structure content. Use cases include:

  • Online courses, which often want to structure material by unit or week. It must be easy to link to quizzes and assignments, and to handle common types of content.
  • Online training, which needs to define sequences of material.
  • Online certification, which needs to require certain items to be done, and sometimes to limit access until prerequisites are met.
  • Import of publisher content or content from other systems that support Common Cartridge (import only at the moment).

Use the action buttons at the top of the page to add content.

How Do I?

Tools that allow you to add content to a page:

  • Add text or HTML content to a page
  • Embed content on page. Most useful for audio, video, Flash. Adds a player to the page.
  • Add content link to a page. Add link to a file or web site, optionally uploading file. Similar to embed, excetp that this adds a link while embed puts the content on the page.
  • Add assignments to a page
  • Add quizzes to a page
  • Add forum topic to a page
  • Add inline questions to a page
  • Allow section for student comments
  • Allow students to add their own pages
  • Add additional pages (e.g. subpages) to a page
  • Upload content from a ZIP file
  • Add external web service using IMS Basic Learning Tool Interoperability protocol

The following functions appear in a separate menu "More tools." It appears only on top-level pages.

  • Add Top-level Page(s)
  • Remove Top-level Page
  • Load file in Common Cartridge Format
  • Save file in Common Cartridge Format

Other functions:

  • Reorganize content
  • Change page settings
